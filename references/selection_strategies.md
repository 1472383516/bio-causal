# 河北卷选择秒杀技巧大全

## 一、单选题13种秒杀法

### 法1: 绝对化排除
```
触发词: 都/全部/一定/绝对/只/唯一/从不/完全/所有/必定
操作: 95%为错误选项
例外: 教材明确写"唯一""全部"的(如胰岛素是唯一的降血糖激素)
```

### 法2: 概念偷换识别
```
看选项中的核心名词是否与题干概念一致
"C₃还原"被偷换成"C₃再生"→不同概念
"净光合"被偷换成"总光合"→不同概念
"基因重组"被偷换成"基因突变"→不同概念
```

### 法3: 因果倒置检测
```
题干: 因为A所以B
选项: 因为B所以A → 因果倒置=错误
"因为ATP消耗增加所以光合增强" vs "因为光合增强所以ATP合成增加"
```

### 法4: 范围扩大/缩小
```
题干范围<选项范围→以偏概全=错误
"原核生物没有线粒体所以不能有氧呼吸"→错(好氧细菌)
"所有光合生物都有叶绿体"→错(蓝藻)
```

### 法5: 必要条件≠充分条件
```
"A是B的必要条件" ≠ "有了A就一定有B"
"CO₂是光合的原料" ≠ "增加CO₂光合一定增强"(还有其他限制)
```

### 法6: 知识域混搭
```
一个选项混合两个不相关的过程→错误
"光合作用的暗反应在线粒体中进行"→混搭(暗反应在叶绿体基质)
```

### 法7: 数字陷阱
```
常考数字: 30-32(有氧ATP), 10-20%(能量传递), 2ⁿ(PCR), 1:1(测交比例)
选项中出现36-38(旧版有氧ATP数)→错误
```

### 法8: "可能"vs"一定"
```
含"可能"→大概率正确(留有余地)
含"一定"→大概率错误(太绝对)
```

### 法9: 教材原话检测
```
看似正确的教材用语→检查是否有微小改动
"暗反应在叶绿体基质中进行"→正确(教材原话)
"暗反应只能在叶绿体基质中进行"→错误(加了"只能")
```

### 法10: 反例法
```
给一个绝对化选项→想一个反例→有反例=错误
"所有酶都是蛋白质"→反例:核酶是RNA→错误
```

### 法11: 数据趋势法
```
含"增加/减少/升高/降低"→看方向是否正确
"CO₂浓度增加→光合速率一定增加"→方向对但"一定"错
```

### 法12: 比较级陷阱
```
A比B更XX→需要A和B确实有比较关系
"ATP比ADP更稳定"→错(ATP含高能键,不稳定)
```

### 法13: 图文不符检测
```
若题干有图→每个选项必须能从图中直接或间接推断
不能从图中得出的选项→错误
```

## 二、多选题4种策略

### 策略1: 必定正确识别
```
教材核心概念原封不动→必定正确
如"线粒体是有氧呼吸的主要场所"→正确
```

### 策略2: 必定错误排除
```
绝对化表述→必定错误(除教材明确写的)
概念混淆→必定错误
因果倒置→必定错误
```

### 策略3: 不确定的逐一因果验证
```
每个选项: 假设它为真→推导因果后果→是否与已知事实矛盾?
不矛盾→可能正确(保留)
矛盾→必定错误(排除)
```

### 策略4: 河北卷多选统计规律
```
近年真题: 正确选项数 2个≈30%, 3个≈50%, 4个≈20%
不要因为选了3个就怀疑→河北卷3正确选项最常见!
```

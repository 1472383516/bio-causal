# 高考备考策略参考（按需加载，不随 SKILL.md 每次加载）

> 以下内容在 SKILL.md 中有简短指针。需要详细的题型分析、提分路径、或考点地图时，从这里读取。

## 📐 题型与分值（2021-2025 真实数据）

| 题型 | 分值 | 答题要求 |
|---|---|---|
| 单选 | 2分×13=26分 | 只写字母 |
| 多选 | 3分×5=15分 | 全对3分，漏选1分 |
| 综合题 | 57-59分 | 填空+简答+实验设计 |
| 难度：容易30% / 中等50% / 难题20% |

## 🎯 命题风格

- 情境新颖（用新素材考旧知识）
- 实验探究（重实验设计+结果分析）
- 图表密集（曲线图/流程图/表格）
- 因果清晰（据图分析，不考死记）
- 术语精确（必须用教材原话）

## 📝 答题规范

| 题型 | 模板 | 必写术语 |
|---|---|---|
| 填空 | 直接写关键词 | 教材原话 |
| 原因分析 | 因为…所以…(因果链2-3环) | 酶/激素/神经递质 |
| 实验步骤 | ①分组→②控制变量→③观测指标 | 等量/适宜/相同且适宜 |
| 预期结果 | 若…则…；若…则… | 显著差异/无显著差异 |

## 🔑 高频术语表（三册全）

### 必修一
- 蛋白质功能：结构蛋白/催化/运输/信息传递/免疫
- 细胞膜结构：流动镶嵌模型
- 光合场所：类囊体薄膜/叶绿体基质
- 呼吸场所：细胞质基质/线粒体基质/线粒体内膜

### 必修二
- 基因本质：有遗传效应的DNA片段
- 中心法则：DNA→RNA→蛋白质
- 变异类型：基因突变/基因重组/染色体变异

### 必修三
- 激素调节：反馈调节/分级调节
- 免疫类型：非特异性免疫/特异性免疫
- 种群数量："J"型增长/"S"型增长

## 📚 60→99 提分路径

### 第一阶段：60→75（保基础，+15分）
| 动作 | 时间 | 提分 |
|---|---|---|
| 背教材原话对照表 | 3天 | +5 |
| 绝对化词排除法 | 1天 | +3 |
| 概念混淆列表对比 | 2天 | +4 |
| 因果链完整训练 | 2天 | +3 |

### 第二阶段：75→85（攻实验，+10分）
教材原话默写(5天,+4)、实验三步模板(3天,+3)、原因分析训练(2天,+3)

### 第三阶段：85→95（冲难题，+10分）
多选满分(2天,+3)、遗传大题(3天,+5)、生态大题(2天,+2)

### 第四阶段：95→99（完美发挥，+4分）
错题本迭代(每天30min,+2)、避坑指南(每周1次,+1)、考前冲刺(考前1周,+1)

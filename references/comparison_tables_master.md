# 河北卷全套知识对比表

> 所有需要"比较异同"的大题考点，一表打尽

## 一、光合vs呼吸完整对比 (12列)

| 维度 | 光合作用 | 细胞呼吸(有氧) |
|------|---------|---------------|
| 场所 | 叶绿体 | 细胞质基质+线粒体 |
| 条件 | 光+色素+酶 | 酶(O₂) |
| 物质变化 | CO₂+H₂O→C₆H₁₂O₆+O₂ | C₆H₁₂O₆+O₂→CO₂+H₂O |
| 能量变化 | 光能→化学能(ATP+NADPH→有机物) | 有机物化学能→热+ATP(30-32个) |
| ATP合成 | 光合磷酸化(光能驱动) | 氧化磷酸化(NADH/FADH₂驱动) |
| [H]来源 | H₂O光解 | 葡萄糖+中间产物+H₂O |
| [H]载体 | NADPH | NADH+FADH₂ |
| [H]去向 | 还原C₃ | 传递给O₂→H₂O |
| O₂ | 产生(来自H₂O) | 消耗(生成H₂O) |
| CO₂ | 消耗 | 产生 |
| 代谢类型 | 同化作用(合成代谢) | 异化作用(分解代谢) |
| 影响因素 | 光强/CO₂/温度/水/矿质 | O₂/温度/水/底物浓度 |

## 二、光反应vs暗反应 (10列)

| 维度 | 光反应 | 暗反应(卡尔文循环) |
|------|--------|-----------------|
| 场所 | 类囊体薄膜 | 叶绿体基质 |
| 条件 | 光+色素+H₂O | CO₂+ATP+NADPH+Rubisco |
| 物质变化 | H₂O→O₂, NADP⁺→NADPH, ADP→ATP | CO₂→C₃→G3P→C₆H₁₂O₆ |
| 能量变化 | 光能→活跃化学能(ATP+NADPH) | 活跃化学能→稳定化学能(有机物) |
| 产物 | O₂, ATP, NADPH | G3P→葡萄糖/淀粉, ADP, Pi, NADP⁺ |
| 与光关系 | 必须光 | 不需光但依赖光反应产物 |
| 关键结构 | 类囊体膜(电子传递链) | 基质(卡尔文循环酶) |
| 影响因素 | 光强,光质,水 | CO₂,温度,ATP/NADPH供给 |
| 阻断后果 | 暗反应停(缺ATP/NADPH) | 反馈抑制光反应(ADP/NADP⁺不再生) |
| 化学渗透 | H⁺泵入类囊体腔 | 无 |

## 三、有氧vs无氧呼吸 (10列)

| 维度 | 有氧呼吸 | 无氧呼吸 |
|------|---------|---------|
| 场所 | 细胞质基质+线粒体 | 细胞质基质 |
| O₂需求 | 需要 | 不需要 |
| 有机物分解 | 彻底(→CO₂+H₂O) | 不彻底(→乳酸或乙醇+CO₂) |
| 产能 | 30-32 ATP/葡萄糖 | 2 ATP/葡萄糖 |
| NAD⁺再生 | 电子传递链(产大量ATP同时) | 丙酮酸→乳酸/乙醇(产少量ATP后) |
| 共同阶段 | 糖酵解 | 糖酵解 |
| 产物(动物) | CO₂+H₂O | 乳酸 |
| 产物(植物/酵母) | CO₂+H₂O | 乙醇+CO₂ |
| ATP来源 | 糖酵解+TCA+氧化磷酸化 | 仅糖酵解 |
| 第二阶段 | 丙酮酸→乙酰CoA→TCA | 丙酮酸→乳酸/乙醇(不产ATP!) |

## 四、DNA复制vs转录vs翻译 (12列)

| 维度 | DNA复制 | 转录 | 翻译 |
|------|--------|------|------|
| 场所(真核) | 细胞核 | 细胞核 | 核糖体(细胞质) |
| 模板 | DNA双链 | DNA一条链(模板链) | mRNA |
| 原料 | 4种dNTP | 4种NTP | 20种氨基酸 |
| 碱基配对 | A-T, G-C | A-U, T-A, G-C | 密码子-反密码子(A-U,G-C) |
| 酶 | DNA聚合酶等 | RNA聚合酶 | 核糖体(含rRNA+蛋白质) |
| 产物 | 2个DNA分子 | 1条mRNA | 1条多肽链 |
| 方向 | 5'→3' | 5'→3' | N端→C端 |
| 是否需要引物 | 是(RNA引物) | 否 | 否 |
| 是否需ATP/GTP | 是 | 是 | 是(GTP) |
| 碱基互补程度 | 全互补 | 全互补 | 3碱基(密码子)互补 |
| 产物去向 | 分配到子细胞 | 出核→细胞质 | 折叠→功能蛋白 |
| 产物数量 | 1个DNA→2个 | 1个基因→多条mRNA | 1条mRNA→多条多肽(多聚核糖体) |

## 五、有丝分裂vs减数分裂 (12列)

| 维度 | 有丝分裂 | 减数分裂 |
|------|---------|---------|
| 发生细胞 | 体细胞 | 原始生殖细胞 |
| 分裂次数 | 1次 | 2次 |
| 子细胞数 | 2 | 4 |
| 染色体数变化 | 2n→4n→2n(不变) | 2n→n(减半) |
| DNA数变化 | 2C→4C→2C | 2C→4C→2C→1C |
| 同源染色体行为 | 无联会/分离 | 有联会/交叉互换/分离 |
| 着丝粒分裂 | 后期 | 减II后期 |
| 染色体减半关键 | 无 | 减I后期(同源染色体分离) |
| 子细胞遗传组成 | 与母细胞相同 | 各不相同(重组) |
| 意义 | 生长/修复/无性生殖 | 有性生殖/遗传多样性 |
| 变异来源 | 基因突变 | 基因突变+基因重组+染色体变异 |
| 秋水仙素作用 | 抑制纺锤体(→四倍体细胞) | 抑制减I或减II纺锤体 |

## 六、体液免疫vs细胞免疫 (8列)

| 维度 | 体液免疫 | 细胞免疫 |
|------|---------|---------|
| 作用对象 | 细胞外病原体/毒素 | 感染细胞/肿瘤细胞/移植细胞 |
| 关键细胞 | B细胞→浆细胞 | Tc细胞(CTL) |
| APCs呈递 | MHC-II类分子 | MHC-I类分子 |
| 识别细胞 | Th(CD4⁺) | Tc(CD8⁺) |
| 效应分子 | 抗体 | 穿孔素+颗粒酶 |
| 效应方式 | 抗体中和/凝集/调理 | 诱导靶细胞凋亡 |
| 记忆细胞 | 记忆B细胞 | 记忆Tc细胞 |
| 疫苗主要激活 | 体液免疫为主 | 需活疫苗或载体 |

## 八、主动运输vs被动运输(自由扩散vs协助扩散) (8列)

| 维度 | 自由扩散 | 协助扩散 | 主动运输 |
|------|---------|---------|---------|
| 方向 | 顺浓度梯度 | 顺浓度梯度 | 逆浓度梯度 |
| 载体 | 不需 | 需要 | 需要 |
| 能量 | 不耗能 | 不耗能 | 耗能(ATP/离子梯度) |

## 九、原核vs真核细胞 (10列)

| 维度 | 原核细胞 | 真核细胞 |
|------|---------|---------|
| 大小 | 1-10μm | 10-100μm |
| 核膜 | 无 | 有 |
| 染色体 | 1条环状DNA(拟核) | 多条线状DNA(细胞核) |
| 细胞器 | 仅有核糖体 | 多种膜性细胞器 |
| 细胞壁 | 肽聚糖(细菌) | 纤维素(植物)/几丁质(真菌)/无(动物) |
| 分裂方式 | 二分裂 | 有丝分裂/减数分裂 |
| 转录翻译 | 偶联(同时同地) | 分隔(核内转录,质内翻译) |
| 基因结构 | 无内含子(多数) | 有内含子+外显子 |
| 核糖体 | 70S | 80S |
| 代表 | 细菌/蓝藻/放线菌/支原体 | 动植物/真菌/原生生物 |

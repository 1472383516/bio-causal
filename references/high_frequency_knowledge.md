## 一、光合作用与呼吸作用（权重 ★★★★★ 15-20%）

### 1.1 光合作用总览

| 知识点 | 核心内容 | 速答模板 |
|--------|---------|---------|
| 光反应场所 | 类囊体薄膜 | "光反应在类囊体薄膜上进行，因为色素和电子传递链位于此处" |
| 暗反应场所 | 叶绿体基质 | "暗反应在叶绿体基质进行，因为Rubisco等酶位于此处" |
| 水的光解 | H₂O → 2H⁺ + 2e⁻ + ½O₂ (PSII) | "水在PSII被光解，产生O₂、H⁺和电子" |
| NADPH生成 | NADP⁺ + 2e⁻ + H⁺ → NADPH (PSI) | "NADP⁺在PSI接收电子被还原为NADPH" |
| ATP合成 | ADP+Pi → ATP (ATP合酶，H⁺梯度驱动) | "H⁺通过ATP合酶回流驱动ADP磷酸化" |
| CO₂固定 | CO₂ + RuBP →(Rubisco)→ 2×3-PGA | "CO₂在Rubisco催化下与RuBP结合生成3-PGA" |
| C₃还原 | 3-PGA →(ATP+NADPH)→ G3P | "3-PGA被ATP磷酸化+NADPH还原生成G3P" |
| RuBP再生 | 5G3P →(3ATP)→ 3RuBP | "5分子G3P消耗3ATP再生3分子RuBP" |

### 1.2 光合关键曲线

```
光补偿点 = 光合速率 = 呼吸速率（净光合=0）
光饱和点 = 光合速率不再随光强增加的临界点
CO₂补偿点 = 光合吸收CO₂ = 呼吸释放CO₂（净光合=0）

速判：
- 曲线与X轴交点 = 补偿点
- 曲线平台起点 = 饱和点
- Y轴截距 = 呼吸速率（光照=0时）
```

### 1.3 光合高频陷阱

| 陷阱 | 正确理解 |
|------|---------|
| "CO₂浓度增加→光合速率一定增加" | 否，达到CO₂饱和点后不再增加 |
| "光照越强光合越好" | 否，光饱和后无效；过强导致光抑制 |
| "RuBP再生需要NADPH" | 否，RuBP再生只需ATP，不需NADPH ← 高频考点！|
| "暗反应不需要光" | 暗反应不需要光但依赖光反应的产物(ATP/NADPH) |
| "净光合速率=总光合速率" | 净=总-呼吸，黑暗测呼吸、光照测净光合 |

### 1.4 细胞呼吸速查

| 阶段 | 场所 | 产物 | ATP |
|------|------|------|-----|
| 糖酵解 | 细胞质基质 | 2丙酮酸 + 2NADH | 净2 |
| 丙酮酸氧化脱羧 | 线粒体基质 | 2乙酰CoA + 2CO₂ + 2NADH | 0 |
| 柠檬酸循环 | 线粒体基质 | 4CO₂ + 6NADH + 2FADH₂ + 2GTP | 2 |
| 氧化磷酸化 | 线粒体内膜 | H₂O | ~26-28 |

**速记**：1葡萄糖 → 30-32 ATP（有氧）；2 ATP（无氧）

### 1.5 光合+呼吸耦合速判

```
白天：光合 > 呼吸 → 净释放O₂、净吸收CO₂ → 有机物积累
夜晚：只有呼吸 → 净吸收O₂、净释放CO₂ → 有机物消耗

关键转折：
- 光补偿点(早晨/傍晚) → 光合=呼吸 → 有机物不增不减
- 午休现象(C3植物) → 高温→气孔关闭→CO₂↓→光合↓
```

## 二、遗传学（权重 ★★★★★ 20-25%）

### 2.1 遗传定律核心

| 知识点 | 核心内容 | 速答模板 |
|--------|---------|---------|
| 分离定律 | 等位基因在减I后期随同源染色体分离 | "一对等位基因，配子含其中一个，比例1:1" |
| 自由组合定律 | 非同源染色体上的非等位基因自由组合 | "两对独立基因的配子，比例1:1:1:1" |
| 连锁交换 | 同源染色体上的非等位基因连锁 | "交换率 = 重组型/(亲本型+重组型)×100%" |

### 2.2 基因互作比例速查（F₂）

| 比例 | 类型 | 速记 |
|------|------|------|
| 9:3:3:1 | 标准独立遗传 | 两对基因无互作 |
| 9:7 | 互补作用 | A和B同时有才显 |
| 9:3:4 | 隐性上位 | aa掩盖B/b |
| 12:3:1 | 显性上位 | A掩盖B/b |
| 15:1 | 叠加作用 | 有A或有B就显 |
| 9:6:1 | 积加作用 | A和B各贡献一部分 |
| 13:3 | 抑制基因 | 某基因抑制另一基因 |
| 1:4:6:4:1 | 数量性状 | 多基因加性效应 |

### 2.3 伴性遗传速判

```
口诀：
- 无中生有→隐性；有中生无→显性
- 男患者远多于女 → X染色体隐性
- 女患者多于男 → X染色体显性
- 仅男患者，父→子→孙 → Y染色体

速判流程（10秒）：
① 找"无中生有"或"有中生无"→判显隐性
② 看男女比例 →判常染色体还是性染色体
③ 从已知推未知 →定基因型
```

### 2.4 基因表达速查

| 步骤 | 场所 | 关键 | 河北陷阱 |
|------|------|------|---------|
| 转录 | 细胞核(真核) | RNA聚合酶，模板链3'→5' | 启动子≠起始密码子 |
| 翻译 | 核糖体 | mRNA上的密码子，tRNA上的反密码子 | 一个mRNA可同时被多个核糖体翻译 |
| 中心法则 | DNA→RNA→蛋白质 | 逆转录(RNA→DNA)仅限某些病毒 | 不是所有生物都有逆转录 |

### 2.5 变异类型速辨

| 类型 | 本质 | 检测方法 | 河北陷阱 |
|------|------|---------|---------|
| 基因突变 | 碱基序列改变 | 基因测序/PCR | 同义突变不改变氨基酸 |
| 基因重组 | 原有基因重新组合 | 杂交实验 | 不是产生新基因！ |
| 染色体结构变异 | 缺失/重复/倒位/易位 | 核型分析 | 倒位→倒位环→部分配子不育 |
| 染色体数目变异 | 非整倍体/整倍体 | 染色体计数 | 秋水仙素抑制纺锤体≠抑制着丝粒分裂 |

|--------|------|------|

## 五、细胞分裂（权重 ★★★☆☆ 5-8%）

| 知识点 | 核心 | 速答 |
|--------|------|------|
| 间期 | DNA复制+蛋白质合成 | 染色体数不变，DNA数加倍 |
| 有丝后期 | 着丝粒分裂，姐妹染色单体分开 | 染色体数暂时加倍(4n→2n) |
| 减I后期 | 同源染色体分离 | 染色体数减半的关键！|
| 减II后期 | 着丝粒分裂 | 与有丝分裂后期相同 |
| 交叉互换 | 减I前期，同源染色体非姐妹染色单体交换 | 基因重组来源之一 |

**速判口诀**：
```
有同源染色体+联会 → 减I
有同源染色体+无联会 → 有丝
无同源染色体 → 减II
```

## 六、生物工程（权重 ★★★★★ 8-12%）

### 6.1 基因工程核心步骤

| 步骤 | 操作 | 速答关键词 |
|------|------|-----------|
| ①获取目的基因 | PCR/化学合成/基因文库 | 引物设计关键 |
| ②构建表达载体 | 质粒+目的基因+标记基因+启动子+终止子 | 限制酶+DNA连接酶 |
| ③导入受体细胞 | 农杆菌/基因枪/显微注射/CaCl₂ | 感受态细胞 |
| ④筛选 | 标记基因→抗性/荧光筛选 | 未导入=无抗性→死亡 |
| ⑤鉴定 | PCR/分子杂交/抗原抗体检测 | 确认目的基因存在+表达 |

### 6.2 PCR速查

| 步骤 | 温度 | 作用 |
|------|------|------|
| 变性 | 95°C | DNA双链→单链(氢键断裂) |
| 退火 | 55°C | 引物与模板互补配对 |
| 延伸 | 72°C | Taq酶合成新链(5'→3') |

**速算**：n个循环 → 2ⁿ倍扩增
**陷阱**：PCR扩增的是两引物之间的片段，不是整个DNA

## 七、实验设计（权重 ★★★★★ 10-15%）

### 7.1 实验设计六要素

```
① 实验目的 → 探究{X}对{Y}的影响
② 自变量 → {操作变量}，设几个梯度？
③ 因变量 → {测量指标}，怎样量化？
④ 控制变量 → 除自变量外其他条件________（填"相同且适宜"）
⑤ 对照组 → 不施加X的组（或施加已知条件的阳性对照）
⑥ 预期结果 → 若X影响Y，则实验组{___}，对照组{___}
```

### 7.2 实验设计高频陷阱

| 陷阱 | 正确做法 |
|------|---------|
| 单一变量没控制 | 除自变量外，各组其他条件必须完全相同 |
| 对照组缺失 | 必须设对照组（空白对照/条件对照/自身对照） |
| 样本量不足 | 每组至少3个重复，取平均值（排除偶然性） |
| 检测指标不合理 | 指标必须可量化、与因变量直接相关 |
| 结论过度推广 | 结论仅限实验条件范围内 |

## 八、河北卷高频综合考点

### 8.1 "基因→性状"完整因果链路

```
基因(DNA) →(转录,RNA聚合酶)→ mRNA →(翻译,核糖体)→ 蛋白质(酶/结构蛋白)
  → 酶催化代谢反应 → 代谢产物变化 → 性状表现

阻断点：
- 基因突变 → 酶失活 → 代谢中断 → 性状改变(如白化病:酪氨酸酶缺陷)
- 环境因素 → 影响酶活性或基因表达 → 性状改变(如温度影响花色)
```

### 8.2 "结构→功能"因果模板

```
{结构特征} → 决定了 → {功能特性} → 导致 → {生理结果}

示例：
- 线粒体内膜折叠成嵴 → 增大表面积 → 容纳更多电子传递链 → 高效ATP合成
- 叶绿体类囊体堆叠成基粒 → 增大膜面积 → 更多色素和电子传递链 → 高效光反应
- 小肠绒毛+微绒毛 → 增大吸收面积 → 高效营养吸收
```

### 8.3 "反馈调节"通用模板

```
负反馈(维持稳定)：
  输出↑ → 抑制上游 → 输出↓ → 恢复 ← 激素调节/体温/血糖/生态

正反馈(加速完成)：
  输出↑ → 促进上游 → 输出↑↑ → 完成 ← 分娩(催产素)/凝血/动作电位
```

## 九、河北卷高频术语速记

| 考题常见问法 | 必须出现的术语 | 常见丢分原因 |
|-------------|---------------|-------------|
| "分析原因" | 因果链必须完整（条件→机制→结果） | 只写结论不写中间机制 |
| "说明理由" | 引用教材原理 + 结合题目条件 | 脱离题目情境空泛答题 |
| "设计实验方案" | 六要素完整 + 预期结果 | 漏对照组或变量控制 |
| "预测结果" | 写出具体数据变化 + 方向 | 只说"增加/减少"不写具体 |
| "评价方案是否合理" | 指出缺陷 + 说明后果 + 提出修改 | 只说不合理不解释为什么 |
| "判断依据" | 从题目数据/现象出发+归因 | 用知识代替题目信息 |
| "得出结论" | 结论仅限实验条件范围内 | 过度推广 |
| "解释现象" | 现象描述+机制归因+原理支撑 | 现象和原因混淆 |
| "比较异同" | 列表对比+分别说明+共同点总结 | 只写不同不写相同 |

## 十、高频表述模板速查

### 原因解释类
```
"因为{条件变化}，导致{机制变化}，从而{结果变化}"
"由于{物质/结构}的{特性}，使得{功能}能够{实现方式}"
"当{自变量}改变时，{中间环节}发生变化，最终体现为{因变量}的改变"
```

### 实验结论类
```
"实验结果表明，{自变量}对{因变量}有{影响方向}作用"
"在{条件范围}内，随{自变量}增加，{因变量}呈{趋势}趋势"
"与对照组相比，实验组{差异}，说明{因果归因}"
```

### 比较异同类
```
"相同点: {共同特征1}、{共同特征2}..."
"不同点: A{特征1}，而B{特征2}；A{特征3}，而B{特征4}"
"本质区别在于{底层机制差异}"
```

## 十一、河北卷必背数字

| 知识点 | 必记数值 | 河北陷阱 |
|--------|---------|---------|
| 1葡萄糖有氧呼吸 | 30-32 ATP | 不要记成36-38！(新版教材修正) |
| 能量传递效率 | 10-20% | 各营养级之间，不是各生物之间 |
| PCR循环n次 | 2ⁿ倍扩增 | 不是n倍！是指数增长 |
| ATP含高能磷酸键 | 2个 | β-γ和α-β之间，α-α不是高能键 |
| DNA碱基互补 | A=T, G≡C | Chargaff规则，用于计算各碱基比例 |
| 减数分裂DNA变化 | 2C→4C→2C→1C | 间期复制→减I→减II |
| 有丝分裂染色体数 | 2n→4n→2n | 后期加倍(着丝粒分裂) |
| 渗透压(mOsm/L) | 血浆≈300 | 等渗溶液0.9%NaCl |

## 十二、选择题秒杀技巧

### 绝对化词语排除法
```
看到"都/全部/一定/绝对/只/唯一/从不"→90%为错误选项
例外需教材原文完全支持

示例：
✗ "所有酶都是蛋白质" → 错(核酶是RNA)
✗ "光合作用一定需要叶绿体" → 错(蓝藻无叶绿体)
✗ "基因突变一定导致性状改变" → 错(同义突变/隐性突变)
```

### 因果倒置识别法
```
正确因果: A(原因)→B(结果)
陷阱: B→A (因果倒置)

示例：
✗ "因为ATP消耗增加，所以光合作用增强" → 因果倒置
✓ "因为光合作用增强，所以ATP合成增加"
```

### 概念偷换识别法
```
原文: 术语A = 定义X
选项: 术语A = 定义Y (偷换)

示例：
✗ "净光合速率 = 总光合速率" → 偷换(净≠总)
✗ "基因重组 = 产生新基因" → 偷换(重组≠突变)
✗ "染色体变异 = 基因突变" → 偷换(结构改变vs碱基改变)
```

## 十三、各专题精华速记卡片 (50张)

### 光合作用10张

**卡1 光合总反应**: 6CO₂+12H₂O→(光能,叶绿体)→C₆H₁₂O₆+6O₂+6H₂O
**卡2 光反应产物**: ATP+NADPH+O₂ | 场所:类囊体薄膜 | 条件:光+色素+H₂O

## 十四、河北卷非选择题满分公式集

### 公式1: 原因解释 (3分)
```
得分=条件(1分)+机制(1分)+结果(1分)
句式: "由于{条件变化}，{中间机制}，因此{结果}"

满分示例: "由于高温导致气孔关闭(1分)，CO₂进入减少使得暗反应速率下降(1分)，因此净光合速率降低有机物积累减少(1分)"
```

### 公式2: 实验设计 (6分)
```
得分=目的(1分)+自变量(1分)+因变量(1分)+控制变量(1分)+对照组(1分)+预期结果(1分)

满分骨架:
(1)实验目的: 探究{X}对{Y}的影响
(2)自变量: 设置{X₁}{X₂}{X₃}三组梯度
(3)因变量: 测定{Z指标}
(4)控制变量: {其他条件}保持相同且适宜
(5)对照组: 不施加X的空白对照组
(6)预期结果: 若X促进Y→实验组Z>对照组Z
```

### 公式3: 曲线分析 (4分)
```
得分=读图(1分)+归因(2分)+推论(1分)

句式:
"由图可知，横轴为{X}纵轴为{Y}(1分)

若{改变某条件}→曲线将{移动方向}(1分)"
```

### 公式4: 评价改进 (3分)
```
得分=缺陷(1分)+后果(1分)+改进(1分)
```

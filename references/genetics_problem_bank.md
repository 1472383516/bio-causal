## 一、九种基因互作比例详解

### 9:3:3:1 — 完全显性自由组合（无互作）

代谢通路:
```
基因A → 酶A → 性状I（独立表达）
基因B → 酶B → 性状II（独立表达）
两条通路各自独立，无交叉干扰。
```

基因型-表型对应表:

| 基因型 | 表型 | 原因 |
|--------|------|------|
| A_B_ (9/16) | 双显性状 | 两条通路均正常运转 |
| A_bb (3/16) | 显I隐II | 仅I通路正常 |
| aaB_ (3/16) | 隐I显II | 仅II通路正常 |
| aabb (1/16) | 双隐性状 | 两条通路均阻断 |

因果链: `A_B_ → 酶A+酶B均正常 → 两条代谢通路完整 → 双显性状`

河北真题示例: 2018年河北卷T32改编，两对独立遗传基因控制豌豆花色与粒形，F2表型比9:3:3:1。

速判口诀: **"九三三一独立算，无互作纯自由组合"**

### 9:7 — 互补效应（双显性互补）

代谢通路:
```
前体物质 →[酶A]→ 中间产物 →[酶B]→ 终产物（显性性状）
A基因编码酶A，B基因编码酶B。两种酶缺一不可。
```

基因型-表型对应表:

| 基因型 | 表型 | 原因 |
|--------|------|------|
| A_B_ (9/16) | 显性 | 酶A+酶B均正常，代谢通路完整 |
| A_bb (3/16) | 隐性 | 缺酶B，通路阻断 |
| aaB_ (3/16) | 隐性 | 缺酶A，通路阻断 |
| aabb (1/16) | 隐性 | 两种酶均缺，通路完全阻断 |

因果链: `A_B_ → 酶A正常+酶B正常 → 前体→中间产物→终产物 → 显性性状；任何一酶缺失 → 终产物不能合成 → 隐性性状`

河北真题示例: 2016年河北卷T30，甜豌豆花色遗传，紫色需同时具备C和P基因，F2紫:白=9:7。

速判口诀: **"九七互补两酶串，断任一环全完蛋"**

### 9:3:4 — 隐性上位效应

代谢通路:
```
前体 →[酶A]→ 中间产物（有色） →[酶B]→ 终产物（另一种颜色）
aa时：酶A缺失 → 前体无法转化为中间产物 → 无论B基因如何，终产物均不合成（aa上位遮盖B_和bb）
```

基因型-表型对应表:

| 基因型 | 表型 | 原因 |
|--------|------|------|
| A_B_ (9/16) | 表型I | 通路完整，终产物正常 |
| A_bb (3/16) | 表型II | 有酶A但缺酶B，中间产物积累 |
| aaB_ (3/16) | 表型III（上位） | 缺酶A，通路在上游阻断 |
| aabb (1/16) | 表型III（上位） | 缺酶A+酶B，同上 |

因果链: `aa存在 → 酶A缺失 → 前体滞留 → 无论B基因型如何 → 均为表型III（aa对上位遮盖）`

河北真题示例: 2019年河北模拟，小鼠毛色遗传（A决定黑色素合成，B决定褐黑素转化），F2黑:棕:白=9:3:4。

速判口诀: **"九三四隐上位，aa一遮整条链"**

### 12:3:1 — 显性上位效应

代谢通路:
```
前体 →[酶A]→ 终产物（性状I）← 此通路独立于B
前体 →[酶B]→ 性状II ← 当A_时此通路被遮盖
A_ 存在时：无论B是什么，均表现为A决定的性状（显性上位）
```

基因型-表型对应表:

| 基因型 | 表型 | 原因 |
|--------|------|------|
| A_B_ (9/16) | 性状I | A_上位遮盖B_ |
| A_bb (3/16) | 性状I | A_上位，B无影响 |
| aaB_ (3/16) | 性状II | A缺失，B正常表达 |
| aabb (1/16) | 性状III | 双隐性 |

因果链: `A_存在 → 酶A活性 → 无论B基因型 → 均表现性状I（显性上位）`

河北真题示例: 2015年河北卷，南瓜果皮颜色遗传，白色对黄色呈显性上位（W_Y_和W_yy均白色），F2白:黄:绿=12:3:1。

速判口诀: **"十二三一显上位，A盖B来不商量"**

### 15:1 — 重复基因（叠加效应）

代谢通路:
```
前体 →[酶A]→ 终产物（显性性状）
前体 →[酶B]→ 终产物（显性性状）
A和B编码功能相同的同工酶，只要任一基因存在即可合成终产物。
```

基因型-表型对应表:

| 基因型 | 表型 | 原因 |
|--------|------|------|
| A_B_ (9/16) | 显性 | 两条通路均通 |
| A_bb (3/16) | 显性 | A基因单独即可 |
| aaB_ (3/16) | 显性 | B基因单独即可 |
| aabb (1/16) | 隐性 | 双缺，无酶可用 |

因果链: `任一显性基因存在 → 至少一种同工酶 → 终产物合成 → 显性性状`

河北真题示例: 2017年河北模拟，荠菜蒴果形状遗传，T1和T2基因功能冗余，F2三角形:卵形=15:1。

速判口诀: **"十五一冗余基因，有A有B都一样"**

### 9:6:1 — 累加效应（双显性协同）

代谢通路:
```
前体（无色） →[酶A]→ 中间产物（浅色） →[酶B]→ 终产物（深色）
A_使前体→中间产物，B_使中间产物→终产物
A和B各贡献一步转化，缺一则停留在中间态
```

基因型-表型对应表:

| 基因型 | 表型 | 原因 |
|--------|------|------|
| A_B_ (9/16) | 深色（II型） | 两步均完成，终产物积累 |
| A_bb (3/16) | 浅色（I型） | 仅完成第一步，中间产物积累 |
| aaB_ (3/16) | 浅色（I型） | 第一步受阻 |
| aabb (1/16) | 无色（0型） | 两步均受阻 |

因果链: `A_B_ → 酶A+酶B均正常 → 前体→中间产物→终产物 → 深色；仅一酶正常 → 通路半途中断 → 浅色`

河北真题示例: 2019年河北联考，南瓜果形遗传（圆球形×长形），A_B_为扁盘形，A_bb和aaB_为圆球形，aabb为长形。

速判口诀: **"九六一累加串联，断一环就减一半"**

### 13:3 — 抑制效应

代谢通路:
```
前体 →[酶A]→ 终产物（显性性状）
基因B（I）编码抑制蛋白，能抑制A基因的表达
当B_存在时 → 抑制蛋白结合A启动子 → A不表达 → 隐性性状
但当A缺失（aa）时，B的抑制效果被遮盖
```

基因型-表型对应表:

| 基因型 | 表型 | 原因 |
|--------|------|------|
| A_B_ (9/16) | 隐性 | B抑制A表达 |
| A_bb (3/16) | 显性 | A正常表达，无抑制 |
| aaB_ (3/16) | 隐性 | A本来就缺失 |
| aabb (1/16) | 隐性 | A缺失 |

因果链: `I_(B_)存在 → 抑制蛋白结合A调控区 → 转录受阻 → 酶A不表达 → 隐性性状`

河北真题示例: 2018年河北一模，家蚕茧色遗传，Y基因决定黄色，I基因抑制Y表达，F2白:黄=13:3。

速判口诀: **"十三三抑制型，I基因一挡A就停"**

### 1:4:6:4:1 — 多基因数量性状（双基因加性效应）

代谢通路:
```
色素合成量 = 显性等位基因数量的函数
每增加一个显性等位基因（A或B），色素沉积量增加一单位
基因型中等位基因组合决定表型梯度
```

基因型-表型对应表:

| 基因型 | 显性等位基因数 | 比例 |
|--------|----------------|------|
| AABB | 4 | 1/16（最深） |
| AABb, AaBB | 3 | 4/16 |
| AAbb, AaBb, aaBB | 2 | 6/16 |
| Aabb, aaBb | 1 | 4/16 |
| aabb | 0 | 1/16（最浅） |

因果链: `显性等位基因数↑ → 酶总量↑ → 色素合成速率↑ → 色素沉积量↑ → 表型梯度加深`

河北真题示例: 2020年河北卷T32改编，小麦粒色遗传（R1/r1, R2/r2），F2红色（4种深度）:白色 = 15:1的高级形式，若细分则呈1:4:6:4:1。

速判口诀: **"一四六一数量性状，数显性等位基因定深浅"**

### 3:1 — 修饰基因型（仅一对基因有效，另一对纯合致死或在特定背景下才表达）

代谢通路:
```
基因A → 酶A → 关键代谢步骤（主效基因）
基因B → 仅在aa背景下表达（修饰基因）
当B为纯合隐性致死时（bb致死），F2呈现修饰后的3:1
```

基因型-表型对应表:

| 基因型 | 表型 | 原因 |
|--------|------|------|
| A_B_ | 显性 | A主效通路正常 |
| A_bb | 致死/无法存活 | bb致死 |
| aaB_ | 隐性 | A缺失 |
| aabb | 致死/无法存活 | bb致死 |

因果链: `bb → 关键发育基因缺失 → 胚胎致死 → 后代中缺bb个体 → 比例扭曲`

河北真题示例: 2008年全国卷（河北用），小鼠毛色遗传，AY基因纯合致死，F2黄色:黑色=2:1（实为修饰后的3:1）。

速判口诀: **"比例扭曲看致死，总数少了比例变"**

## 二、伴性遗传六种模式

### X染色体隐性遗传（XR）

典型系谱特征:
- 母患子必患（交叉遗传的核心表现）
- 女患父必患
- 系谱中男性患者远多于女性
- 女性患者的所有儿子必定患病
- 可隔代遗传（外祖父→女儿→外孙）

男女患病比例:
- 男性: X^aY → 发病率 = 致病基因频率p
- 女性: X^aX^a → 发病率 = p²
- 男性发病率 >> 女性发病率（若p=0.01，男1%，女0.01%）

交叉遗传特点:
```
外祖父(X^aY) → 女儿(X^AX^a) → 外孙(X^aY)
      ↓                    ↓
   致病X传给女儿      女儿携带者传致病X给儿子
   （父亲→女儿必然）  （母亲→儿子50%概率）
```

速判流程(3步):
1. 找"母患子必患"模式：若某患病女性，其所有儿子均患病 → XR概率高
2. 找"女患父必患"模式：若某患病女性，其父亲患病 → XR概率高
3. 排除法：若出现"父患女必患"→ 不是XR，而是XD

河北常见陷阱:
- 陷阱①：X连锁隐性偶见女性患者，需确认她父亲是否患病
- 陷阱②："交叉遗传"不等于"隔代遗传"——交叉遗传强调性别交叉，隔代仅指跳过一代
- 陷阱③：X^AX^a × X^AY的后代中，儿子患病≠所有儿子患病（需考虑概率50%）

例题+解析:

**例题**: 红绿色盲系谱图中，III-1为色盲男性，III-2为正常男性，II-1正常女性，II-2色盲男性，I-1正常女性，I-2正常男性。判断遗传方式并计算IV-1的患病概率。

**解析**:
- 判断遗传方式: III-1的父亲II-2色盲 → 父亲→儿子没有传递 → 排除Y连锁；III-1的母亲II-1正常 → 母亲是携带者 → 支持XR
- 因果推导: `X^b基因 → 视锥细胞视蛋白异常 → 红绿光感受障碍 → 色盲`
- 确定基因型: II-2为X^bY，II-1必为X^BX^b（因为生了色盲儿子III-1），I-1为X^BX^b（因为生出色盲儿子II-2），I-2为X^BY
- 概率计算: III-1 = X^bY（确定），IV-1若为儿子 → 若III-1妻子为携带者则50%患病

### X染色体显性遗传（XD）

典型系谱特征:
- 父患女必患（关键诊断标志）
- 母患子50%患、女50%患（与XR不同）
- 系谱中女性患者多于男性（约2:1）
- 连续遗传，少见隔代
- 女性杂合子即可患病（与XR需纯合子不同）

男女患病比例:
- 男性: X^DY → 发病率 = p
- 女性: X^DX^d或X^DX^D → 发病率 = p² + 2p(1-p) ≈ 2p（p小时）
- 女性发病率 ≈ 2倍男性发病率

交叉遗传特点:
```
患病父亲(X^DY) → 所有女儿(X^DX^d) → 女儿的孩子各有50%概率
      必然传给女儿          不传给儿子
```

速判流程(3步):
1. 找"父患女必患"模式 → XD强烈信号
2. 找"母患子不一定患" → 排除XR（XR中母患子必患）
3. 看连续遗传 → 排除常显（常显父患不一定女患）

河北常见陷阱:
- 陷阱①：把"父患女必患"当作常染色体显性——常显中父患女患概率仅1/2
- 陷阱②：抗维生素D佝偻病(XD)与软骨发育不全(AD)表型相似，注意区分
- 陷阱③：若系谱中女性患者远多于男性，优先考虑XD

例题+解析:

**例题**: 抗维生素D佝偻病系谱，II-3为患病女性，II-4为正常男性，他们的儿子III-3正常，女儿III-4患病。判断遗传方式。

**解析**:
- 判断: II-4正常男性 × II-3患病女性 → 女儿患病儿子正常 → 若为XD则II-3为X^DX^d → 女儿X^DX^d(患病)，儿子X^dY(正常) ✓
- 因果推导: `X^D → PHEX基因突变 → FGF23过量表达 → 肾小管磷重吸收↓ → 低磷血症 → 骨矿化不全 → 佝偻病`
- 验证: 若为常显 → 女儿患病概率1/2 → 无法确定；但结合群体女性患者更多 → 支持XD

### Y染色体遗传（全雄遗传/限雄遗传）

典型系谱特征:
- 仅男性患病，女性不患病也不携带
- 父传子、子传孙，世代传递不中断
- 一旦出现女性患病，排除Y遗传
- SRY基因决定睾丸发育，相关基因突变影响精子发生

男女患病比例:
- 男性: 携带致病Y染色体者100%患病
- 女性: 0%（无Y染色体）
- 系谱中患者均为男性，呈严格父系传递

交叉遗传特点:
```
无交叉遗传。Y染色体不重组（仅PAR区例外，但致病基因多在NRY区）
患病父亲 → 所有儿子(Y染色体完整传递) → 所有孙子
```

速判流程(3步):
1. 检查是否100%男性患病 → 是则进入步骤2
2. 检查是否严格父传子（无母传子） → 是则进入步骤3
3. 检查是否有女性患病 → 无 → 确诊为Y连锁

河北常见陷阱:
- 陷阱①：外耳道多毛症是Y连锁，但不是所有毛发性状都是Y连锁
- 陷阱②：若系谱中男性远多于女性但偶见女性 → 可能是性连锁+外显率不完全
- 陷阱③：Y连锁≠伴性遗传——"伴性"通常指X连锁

例题+解析:

**例题**: 某家系中所有男性均有外耳道多毛（长硬毛从耳轮边缘长出），所有女性均无此特征，追溯五代均有此特征。

**解析**:
- 判断: 五代均父传子 → 严格Y连锁
- 因果推导: `Y染色体上毛囊发育调控基因 → 毛囊对雄激素超敏 → 外耳道毛囊过度发育 → 多毛症`
- 遗传风险: 男性后代100%患病，女性后代0%

### 从性遗传（Sex-limited Inheritance）

典型系谱特征:
- 基因在常染色体上，但表达受性激素调控
- 杂合子在两性中表型不同（同一基因型在不同性别中表型不同）
- 系谱中两性患病率有差异，但不呈现X连锁模式
- 典型例：早秃（男性杂合子即脱发，女性纯合子才脱发）

男女患病比例:
- 取决于杂合子在两性中的外显率差异
- 例如：早秃 — 男性B_b即脱发，女性需bb才脱发
- 男性发病率 >> 女性发病率

交叉遗传特点:
```
无交叉遗传（常染色体基因）
但表型传递模式可能看似"父传子不传女" → 实际是两性表型标准不同
```

速判流程(3步):
1. 排除X连锁和Y连锁 → 常染色体基因
2. 比较同基因型杂合子在不同性别中的表型
3. 若杂合子表型因性别而异 → 从性遗传

河北常见陷阱:
- 陷阱①：男性早秃与X连锁混淆——男性患者多≠X连锁
- 陷阱②：把从性遗传当作"限性遗传"——限性遗传指性状只在一种性别中表现
- 陷阱③：计算子代概率时需分性别分别计算

例题+解析:

**例题**: 人类早秃，B为早秃基因（杂合子在男性中表现早秃，女性中不表现），b为正常。Bb男性与Bb女性婚配，子代表型。

**解析**:
- 基因型: Bb × Bb → 后代: 1/4 BB, 1/2 Bb, 1/4 bb
- 分性别讨论:
  - 儿子: BB(早秃), Bb(早秃), bb(正常) → 3/4早秃, 1/4正常
  - 女儿: BB(早秃), Bb(正常), bb(正常) → 1/4早秃, 3/4正常
- 因果推导: `B基因 → 毛囊雄激素受体亲和力↑ → 5α-还原酶活性↑ → 睾酮→DHT转化↑ → 毛囊微型化 → 脱发；女性因雌激素拮抗 → 杂合子不表现`

### 限性遗传（Sex-limited Inheritance）

典型系谱特征:
- 性状只在一种性别中表达
- 基因可在常染色体或性染色体上，但表型受性别限制
- 例如：子宫发育异常（仅在女性表达，男性基因型未知）
- 又例如：精子生成障碍（仅在男性表达）

男女患病比例:
- 一性100%不表现该性状
- 另一性按基因型规律表现
- 基因频率在全体中正常分布

交叉遗传特点:
```
无交叉遗传。
即使基因在X染色体上，因性别限制也不表现出交叉遗传模式
```

速判流程(3步):
1. 观察性状是否仅限一性 → 从性或限性
2. 观察该性别中是否有"不患病但后代患病" → 若有则为限性（基因存在但不表达）
3. 区分从性与限性: 从性=两性均可表达但标准不同；限性=一性完全不表达

河北常见陷阱:
- 陷阱①：把限性与Y连锁混淆——Y连锁是基因在Y上，限性是基因可在任何位置但仅在男性表达
- 陷阱②：计算限性遗传概率时，需考虑"不表现性状的个体可能是携带者"

例题+解析:

**例题**: 某性状仅出现在男性中，表现该性状的男子与正常女子结婚，儿子中1/2表现该性状，女儿均正常。判断遗传方式。

**解析**:
- 假设①: Y连锁 → 父传子100%（1/2 ≠ 100%） → 排除
- 假设②: XR → 男性患者(X^aY) × 正常女性(X^AX^A) → 儿子必不患病(A来自母) → 排除
- 假设③: 常显限性（仅在男性表达） → 父(Aa) × 母(aa) → 子代1/2为Aa，但仅在儿子中表现 ✓
- 因果推导: `仅男性表达的基因 → 精子发生关键蛋白 → 缺失导致不育表型 → 女性无对应器官，不表现`

### 细胞质遗传（母系遗传）

典型系谱特征:
- 母患子必患（所有子女均患病，不论性别）
- 父患子不患（父亲性状不传给任何子女）
- 系谱中所有个体通过母亲追踪
- 线粒体DNA/叶绿体DNA的遗传模式
- 不符合孟德尔分离比

男女患病比例:
- 无性别差异（男女均等概率患病）
- 患病与否取决于母亲，与父亲无关
- 若母亲100%线粒体突变 → 所有后代100%受影响（可能有阈值效应）

交叉遗传特点:
```
无交叉遗传。仅通过卵细胞质传递。
精子线粒体在受精后被泛素标记降解 → 合子线粒体完全来自卵细胞。
```

速判流程(3步):
1. 检查是否"母患子皆患"（不分性别） → 是
2. 检查是否"父患病不影响子代" → 是
3. 与XR鉴别的关键: XR中"母患子必患"但女儿不一定（女儿50%），而细胞质遗传是所有子女均患

河北常见陷阱:
- 陷阱①：把XR的"母患子必患"误判为细胞质遗传 → 区分点在于女儿是否也100%患病
- 陷阱②：线粒体疾病有"阈值效应" → 杂胞质中突变比例达到阈值才发病，导致遗传模式不完全规律
- 陷阱③：Leber遗传性视神经病变（LHON）的"男多女少" → 不完外显+性别修饰，不是伴性遗传

例题+解析:

**例题**: 某系谱中，I-1患病女性与I-2正常男性婚配，生育三个孩子全部患病（两男一女）；II-2（患病男性）与II-3（正常女性）婚配，生育四个孩子全部正常。判断遗传方式。

**解析**:
- 判断: I-1患病 → 所有子女100%患病（母患子皆患）；II-2患病但配偶正常 → 所有子女正常（父患不影响子代）
- 确认: 符合细胞质遗传（线粒体遗传）
- 因果推导: `母亲线粒体DNA突变 → ATP合酶亚基缺陷 → 氧化磷酸化障碍 → 能量代谢异常 → 多系统疾病`
- 排除: 若为XR → II-2(X^aY) × II-3(X^AX^A) → 子代男性必正常(来自母亲的X^A) ✓，但女儿必为携带者 → 不符合"全部正常"

## 三、系谱图专题

### 系谱图1 — 基础级: 常染色体隐性

图示描述:
```
I-1(□正常) × I-2(○正常) → II-1(●患病男性), II-2(○正常女性)
II-1(●) × II-3(○外源正常) → III-1(●患病男), III-2(○正常女)
```

遗传方式判断:
- 假设①: 常染色体隐性（双亲正常生出患病儿） → `I-1(Aa) × I-2(Aa) → II-1(aa) → √`
- 假设②: 常染色体显性 → 父母均正常但儿患病，排除
- 结论: **常染色体隐性遗传（AR）**

关键个体基因型推导:
- I-1 = Aa, I-2 = Aa（必为杂合子，因为生出了aa后代）
- II-1 = aa（已知患病）
- II-2: 正常个体来自Aa × Aa → 她可能是AA或2/3概率为Aa
- III-1 = aa（已知患病）
- III-2: 正常个体，取决于II-1(aa) × II-3(外来个体)

概率计算:
- II-2为携带者概率 = 2/3（已知正常，排除aa后的基因型比）
- 若II-2(Aa)与外源(Aa)结婚 → 子代患病概率 = 2/3 × 1/4 = 1/6

河北变式: 增加"致病基因频率p" → 计算外源个体为携带者的概率（2pq ≈ 2p） → 患病概率为2/3 × 2p × 1/4

### 系谱图2 — 中等级: X染色体隐性

图示描述:
```
I-1(●患病男性) × I-2(○正常女性) → II-1(○正常女), II-2(●患病男), II-3(○正常女), II-4(□正常男)
II-1(○) × II-5(□外来正常) → III-1(●患病男), III-2(□正常男)
II-2(●) × II-6(○外来正常) → III-3(○正常女), III-4(□正常男)
```

遗传方式判断:
- 假设①: X染色体隐性 → I-1患病但儿子II-2患病（父→子不能） → I-1基因型X^aY → 传给女儿的X^a → II-1和II-3为携带者X^AX^a → 她们的儿子可能患病 → ✓
- 假设②: 常染色体隐性 → I-1(aa) × I-2(Aa) → II-1和II-3为100%携带者, II-2可能为患者 → 但II-2男性患病，若为AR则II-2为aa需要I-2也是Aa → 但I-2正常 → 只能算概率 → 不完全排除；但结合III-1(男性患者,母亲II-1正常) → 若为AR需II-1为携带者且配偶也为携带者 → 概率太低 → 倾向XR
- 结论: **X染色体隐性遗传（XR）**

关键个体基因型推导:
- I-1 = X^aY（患病男性）
- I-2 = X^AX^A 或 X^AX^a（若儿子II-2患病则为X^AX^a）
- II-1 = X^AX^a（父亲给X^a，母亲给X^A）
- II-2 = X^aY（已知患病）
- II-1 × II-5: II-1为X^AX^a, II-5为X^AY → 儿子患病概率 = 1/2
- III-1 = X^aY

概率计算:
- III-1的患病概率（已知患病后倒推母亲基因型） → 确认II-1必为携带者
- II-2 × II-6 → 若II-6为正常纯合 → 女儿100%携带者，儿子正常

河北变式: 给出"散发"案例（家系中仅一例患病） → 需考虑新发突变概率 → 母系为携带者或新发突变 → 复发风险不同

### 系谱图3 — 进阶: X染色体显性

图示描述:
```
I-1(□正常男性) × I-2(●患病女性) → II-1(●患病女性), II-2(●患病男性), II-3(□正常男性), II-4(○正常女性)
II-1(●) × II-5(□外来正常) → III-1(●患病女), III-2(□正常男)
II-2(●) × II-6(○外来正常) → III-3(●患病女), III-4(□正常男)
```

遗传方式判断:
- 假设①: X染色体显性 → I-1(X^dY) × I-2(X^DX^d) → 女儿患病(X^DX^d) ✓, 儿子50%患病 ✓ → 验证通过
- 假设②: 常染色体显性 → I-1(aa) × I-2(Aa) → 子女各50%患病 → 但观察II-1(女儿)和II-2(儿子)患病比例无法区分 → 看II-2(患病男)嫁娶 → 若XD则男患女儿100%患病 → 若AD则男患女儿50%患病 → 观察III-3患病 → 若有多个女儿均患病则支持XD
- 假设③: 若II-2患病男性 × II-6正常女性 → 所有女儿患病 → 确诊XD
- 结论: **X染色体显性遗传（XD）**

关键个体基因型推导:
- I-2 = X^DX^d（杂合子患病，因为后代中有正常儿子II-3）
- II-2 = X^DY（患病男性）
- II-2 × II-6: X^DY × X^dX^d → 女儿X^DX^d(100%患病)，儿子X^dY(100%正常)
- III-3 = X^DX^d（必然患病）

概率计算:
- II-2 × II-6: 女儿100%患病，儿子0%患病
- II-1(X^DX^d) × II-5(X^dY): 女儿50%患病，儿子50%患病

河北变式: 增加"不完全外显"条件 → 女性杂合子85%外显 → 计算表型概率时乘以0.85

### 系谱图4 — 高难: 伴性致死

图示描述:
```
I-1(□正常男性) × I-2(○正常女性) → II-1(○正常女), II-2(□正常男), II-3(□正常男), II-4(□早夭男婴)
II-1(○) × II-5(□外来正常) → III-1(●患病男), III-2(□正常男), III-3(○正常女)
III-1(●患病长大) × III-4(○外来正常) → IV-1(□早夭男婴=1), IV-2(□正常男=2), IV-3(○正常女=2)
```

遗传方式判断:
- 假设①: X染色体隐性伴男性胚胎致死 → 患病母亲(X^AX^a)可能生出部分男性早夭 → 系谱中男性早夭+女性正常但可能携带 → ✓
- 假设②: X染色体显性致男性致死 → 患病母亲杂合子 → 儿子50%死亡 → 系谱中男性患者稀少且早夭 → ✓
- 关键区分: 早夭是"存活但不能生育"还是"胚胎期死亡" → 影响系谱中男性患者的有无
- 结论: **X染色体显性遗传，男性半合子致死（男性患者仅存杂合母亲的儿子）**

关键个体基因型推导:
- I-2 = X^AX^a（正常但有致死基因，儿子II-4早夭）
- II-1 = X^AX^a（50%概率携带，父亲I-1给X^A）
- III-1 = X^aY（存活患者，因为母亲是携带者）
- III-1 × III-4 → IV代 → 男婴50%概率致死不育

概率计算:
- II-1携带概率 = 1/2（母亲I-2 X^AX^a → 女儿II-1 50%得到X^a）
- 若II-1为携带者 → III-1患病概率 = 1/2（儿子）
- III-1若存活 → 其子代男性仍有50%致死

河北变式: "男性出生率偏低且伴多系统异常" → 提示X连锁致死+杂合女性表型 → 色素失禁症(Incontinentia pigmenti)

II-1(○色盲) × II-4(□多指) → III-1(□色盲+多指), III-2(□色盲), III-3(□多指), III-4(□正常)

## 四、基因序列图专题（河北特色）

### 类型1: 启动子区变异——转录调控分析

典型题目:
某基因5'调控区序列（-200~+1）中有TATA box(-30~-25)、GC box(-90~-80)、CAAT box(-75~-70)。现发现-29位T→G突变，和-85位C→T突变。分析两种突变对基因表达的影响。

模板链识别:
```
编码链(正义链): 5'-...TATAAAA...GGGCGG...CCAAT...ATG...-3'
模板链(反义链): 3'-...ATATTTT...CCCGCC...GGTTA...TAC...-5'
转录方向: 从模板链3'→5'合成mRNA 5'→3'
```

关键元件标注:
- **TATA box** (-30~-25, TATAAAA): RNA聚合酶II识别结合位点, TFIID(TBP亚基)识别序列
- **GC box** (-90~-80, GGGCGG): 转录因子Sp1结合位点
- **CAAT box** (-75~-70, CCAAT): NF-Y/CTF转录因子结合位点
- **起始密码子** (+1~+3, ATG): 翻译起始
- **终止密码子**: 在编码区3'端(TAA/TAG/TGA)

变异影响分析:
- -29位T→G(在TATA box内): `TATAAAA → TAGAAAA` → TBP识别亲和力↓ → TFIID结合效率↓ → RNA Pol II招募↓ → 转录水平降低70-90% → 表达量↓
- -85位C→T(在GC box上游): 不在核心调控元件内 → 可能不影响或轻微影响(若在Sp1结合位点则亲和力↓)

因果链总览:
```
启动子突变 → 转录因子结合效率↓ → RNA Pol II不能有效招募 → 转录速率↓ → mRNA量↓ → 蛋白表达↓ → 表型↓
```

### 类型2: 编码区点突变——同义/错义/无义

典型题目:
某基因编码区部分序列（+157~+201）:
```
野生型: 5'-ATG TCG GGA TGG AAG CCA TTC TTA CAG TAA-3'
突变型: 5'-ATG TCG GGA TGA AAG CCA TTC TTA CAG TAA-3'
```
分析突变类型并预测表型。

关键元件标注:
- 起始密码子: ATG(+1~+3), 位于上游
- 突变位点: +169 位置G→A转换
- 终止密码子: TGA（新产生）, 或TAA（原有，+181~+183）
- 原终止密码子: TAA(+181~+183)

变异影响分析（按突变类型分类）:
- **同义突变**: 密码子改变但氨基酸不变 → CGA(CGG/CGT/CGC)→Arg → 无表型影响
- **错义突变**: 密码子改变，氨基酸替换 → TGG(Trp)→TGA(Stop) ≠ 错义（是终止）; 若为TGG→CGG(Trp→Arg)则为错义 → 取决于氨基酸理化性质差异
    - 保守替换（同性质氨基酸）: 功能影响小
    - 非保守替换（异性质氨基酸）: 蛋白质构象改变 → 功能受损
- **无义突变**: 产生终止密码子 → TGG(Trp) → TGA(Stop) → 翻译提前终止 → 截短蛋白
    - 若终止在蛋白N端 → 完全无功能（无义介导的mRNA降解NMD）
    - 若终止在C端近 → 可能保留部分功能
- **移码突变**: 插入/缺失非3倍数的碱基 → 阅读框改变 → 下游全部氨基酸错译 → 通常产生提前终止 → 完全无功能

本题分析:
野生型: Trp-Lys-Pro-Phe-Leu-Gln-Stop
突变型: 第5个密码子TGG→TGA(Stop) → 蛋白截短，缺少Lys-Pro-Phe-Leu-Gln → 大概率无功能
突变类型: **无义突变（点突变→提前终止）**

因果链总览:
```
点突变(G→A) → 密码子TGG→TGA → 翻译至第5位终止 → 截短蛋白(缺失下游功能域) → NMD途径降解mRNA → 蛋白缺失 → 功能丧失 → 隐性遗传病
```

### 类型3: 缺失/插入突变——移码与框内

典型题目:
某基因部分编码序列（+100~+160）:
```
野生型: 5'-...GAC TTC CAG AAA GGA...-3' (Asp-Phe-Gln-Lys-Gly)
突变型1: 5'-...GAC TTC CAG AAA GGA...-3' 中缺失+124位C 
突变型2: 5'-...GAC TTC CAG AAA GGA...-3' 中缺失+121~+123位TTC
```
预测两种突变的蛋白质后果。

变异影响分析:
- **缺失3的倍数（框内缺失）**: 丢失完整氨基酸 → 仅缺失该位氨基酸，阅读框不变
    - 例: 缺失TTC(Phe) → 蛋白: Asp-Gln-Lys-Gly → 可能部分保留功能
- **缺失非3的倍数（移码突变）**: 阅读框改变 → 下游氨基酸序列全变
    - 例: 缺失C → 阅读框+1 → Asp-Phe-Arg-Lys-Glu→新的终止位点 → 截短或异常延伸

突变型1分析(缺失C):
原始: GAC|TTC|CAG|AAA|GGA → Asp-Phe-Gln-Lys-Gly
缺失C后: GAC|TTC|AGA|AAG|GA... → Asp-Phe-Arg-Lys-Glu-... → 移码突变 → 下游改变 → 大概率失活

突变型2分析(缺失TTC):
原始: GAC|TTC|CAG|AAA|GGA → Asp-Phe-Gln-Lys-Gly
缺失TTC后: GAC|CAG|AAA|GGA → Asp-Gln-Lys-Gly → 框内缺失 → 仅缺失Phe → 可能保留功能

因果链总览:
```
缺失非3倍数 → 阅读框移码 → 下游密码子全变 → 新终止密码子出现 → 截短/异常蛋白 → 功能丧失(NMD降解)
缺失3倍数   → 缺失完整氨基酸 → 阅读框不变 → 蛋白缩短但其余序列正确 → 部分功能保留或丧失(取决于缺失位置)
```

### 类型4: 酶切位点与限制性片段长度多态性(RFLP)

典型题目:
某基因包含一个EcoRI酶切位点GAATTC（+500~+505）。正常个体基因PCR扩增产物长度为2kb，酶切后产生1.2kb和0.8kb两个片段。某患者该位点突变为GAATTC→GAAATC，消融酶切位点。请设计PCR-RFLP检测方案并预测电泳结果。

PCR引物设计:
```
上游引物(F): 5'-{基因+400~+420位序列}-3'
下游引物(R): 5'-{基因+2380~+2400位反向互补序列}-3'
扩增片段: 约2000bp(2kb)
酶切位点: GAATTC → +500~+505
```

电泳结果预测:

| 基因型 | PCR产物长度 | 酶切后条带 | 判断 |
|--------|------------|-----------|------|
| 正常纯合(+/+) | 2000bp | 1200bp + 800bp | 2条带 |
| 杂合(+/m) | 2000bp | 2000bp + 1200bp + 800bp | 3条带 |
| 突变纯合(m/m) | 2000bp | 2000bp(未切开) | 1条带 |

因果链总览:
```
点突变消融酶切位点 → 限制酶不能识别GAATTC → 该位点不被切割 → PCR产物保持2000bp整 → 电泳出现高分子量条带 → 与正常对照对比 → 判断基因型
```

河北常见RFLP考题变式:
- 多态位点连锁分析：用多个酶切位点的RFLP组合进行单倍型分析
- 酶切位点"获得"突变：突变产生新的酶切位点 → 正常个体不被切，突变体被切

### 类型5: 可变剪接——外显子跳跃与内含子保留

典型题目:
某基因包含4个外显子(E1-E2-E3-E4)，剪接供体位点为GT，分支点A，剪接受体位点为AG。现发现内含子2的5'剪接位点由GT→AT突变，和内含子3的3'剪接位点由AG→AA突变。预测可变剪接产物。

关键元件标注:
```
正常剪接元件:
E1|GT...A...(T/C)nNCAG|E2|GT...A...(T/C)nNCAG|E3|GT...A...(T/C)nNCAG|E4
   ↑供体          ↑受体    ↑供体          ↑受体   ↑供体          ↑受体

突变:
内含子2供体位点 GT→AT: 剪接体U1 snRNA不能碱基配对 → 5'剪接位点失活
内含子3受体位点 AG→AA: 剪接体U2AF不能识别 → 3'剪接位点失活
```

变异影响分析(剪接产物预测):

**突变①: 内含子2的5'剪接位点GT→AT**
- 剪接体不能识别内含子2的5'端 → 外显子2+内含子2+外显子3作为一个"外显子"保留 → 可能产生:
    - 隐蔽剪接位点激活 → 产生异常转录本
    - 内含子保留 → 翻译至内含子内终止 → 截短蛋白

**突变②: 内含子3的3'剪接位点AG→AA**
- 剪接体不能识别内含子3的3'端 → 外显子3被跳过 → E2直接连接到E4 → 框内或移码缺失

**组合效应分析**:
- 正常剪接: E1→E2→E3→E4 → 全功能蛋白
- E3跳跃(突变②): E1→E2→E4 → 若E3长度非3倍数 → 移码 → 截短; 若E3长度是3倍数 → 框内缺失 → 部分功能
- 内含子2保留(突变①): E1→E2→内含子2→E3→E4 → 内含子内含终止 → 截短蛋白

因果链总览:
```
剪接位点突变 → 剪接体识别异常 → 可变剪接产物 → mRNA序列改变 → 蛋白结构域缺失/截短 → 功能改变/丧失
```

河北真题变式:
- 地中海贫血中的β-珠蛋白基因IVS-II-1 G→A突变（内含子2剪接位点突变）→ β-珠蛋白缺失 → β-地中海贫血
- SMN1/SMN2基因的外显子7跳跃 → 脊髓性肌萎缩症(SMA)

## 五、遗传学计算速算公式

### 5.1 基础公式

| 公式 | 说明 | 使用条件 |
|------|------|----------|
| 配子类型数 = 2ⁿ | n = 杂合基因对数 | 独立遗传 |
| 子代基因型种类 = 3ⁿ | 完全显性，独立遗传 | F2代AaBbCc × AaBbCc |
| 子代表型种类 = 2ⁿ | 完全显性，独立遗传 | 无互作 |
| 测交后代基因型种类 = 2ⁿ | AaBbCc × aabbcc | 独立遗传 |

### 5.2 概率计算公式

**特定基因型概率（独立遗传）:**
```
公式: P(目标基因型) = Π(i=1 to n) P(第i对基因的目标等位基因组合)

例: AaBbCc × AaBbCc → 子代AABbcc概率
= P(AA) × P(Bb) × P(cc)
= 1/4 × 1/2 × 1/4
= 1/32
```

**至少一种显性性状概率:**
```
P(至少一个显性性状) = 1 - P(全部隐性)
例: 三对独立基因 → P = 1 - (1/4)³ = 1 - 1/64 = 63/64
```

### 5.3 连锁与交换公式

| 公式 | 说明 |
|------|------|
| 交换率(重组率) = 重组型配子数 / 总配子数 | 0 ≤ r ≤ 0.5 |
| 基因间遗传距离(cM) = 交换率 × 100 | 1% = 1cM = 1个图距单位 |
| 亲本型配子频率 = (1-r)/2 每种 | 相引相 |
| 重组型配子频率 = r/2 每种 | 相引相 |
| 相斥相交换率的检测: 测交 = 重组型数/总数 | AB/ab × ab/ab |
| 双交换率 = 相邻区间交换率之积 × 符合系数 | 三基因定位 |

**三点测交定位顺序:**
1. 找出数量最少的两种重组型（双交换产物）
2. 与亲本型比较，确定中间的基因
3. 计算两两基因间的遗传距离
4. 绘制连锁图: 基因A —d1— 基因B —d2— 基因C

### 5.4 群体遗传学公式

| 公式 | 说明 |
|------|------|
| p + q = 1 | Hardy-Weinberg等位基因频率 |
| p² + 2pq + q² = 1 | Hardy-Weinberg基因型频率 |
| 携带者频率 = 2pq ≈ 2q(q<<1) | XR中女性携带者频率 |
| 男性发病率 = q | XR中性频率 |
| 女性发病率 = q² | XR中女性频率 |
| 突变-选择平衡: q = μ/s | μ=突变率, s=选择系数 |

### 5.5 系谱概率速算

**Bayes定理在系谱中的应用:**
```
后验概率 = 先验概率 × 条件概率 / Σ(先验概率 × 条件概率)

例: 某女性有2个正常儿子，她是携带者的后验概率:
先验: P(携带者) = 1/2(母亲是携带者)
条件: 若她是携带者 → 生2个正常儿子的概率 = (1/2)² = 1/4
      若她不是携带者 → 生2个正常儿子的概率 = 1² = 1

后验P(携带者|2个正常儿子) = (1/2 × 1/4) / (1/2 × 1/4 + 1/2 × 1)
                           = (1/8) / (1/8 + 1/2)
                           = 1/5
```

### 5.6 特殊遗传模式概率调整

| 遗传模式 | 表型比例 | 概率计算公式 |
|----------|----------|-------------|
| 互补(9:7) | A_B_:其他 = 9:7 | P(显性) = 9/16, P(隐性) = 7/16 |
| 隐性上位(9:3:4) | A_B_:A_bb:aa__ = 9:3:4 | P(A_B_) = 9/16, P(A_bb) = 3/16, P(aa__) = 4/16 |
| 显性上位(12:3:1) | A___:aaB_:aabb = 12:3:1 | P(A___) = 12/16, P(aaB_) = 3/16, P(aabb) = 1/16 |
| 重叠(15:1) | 至少一显:双隐 = 15:1 | P(显性) = 15/16, P(隐性) = 1/16 |
| 抑制(13:3) | 被抑制:不被抑制 = 13:3 | P(被抑制) = 13/16, P(表达) = 3/16 |
| 致死修饰 | 注意调整分母 | 如AY黄色鼠: AYA × AYA → AYAY(致死):AYA(黄):AA(黑) = 0:2:1 → 2:1 |
| 从性遗传 | 分性别计算 | P(男患病)与P(女患病)分别计算 |

## 六、遗传学实验设计模板

### 6.1 验证基因分离定律

#### 方法A: 测交法
**原理**: 杂合子与隐性纯合子杂交，子代表型比应为1:1

**实验步骤:**
1. 选取F1杂合子个体(Aa) × 隐性纯合子(aa)
2. 统计后代表型及其比例
3. 若子代表型比接近1:1 → 支持分离定律

**因果链:**
```
F1(Aa)减数分裂 → 同源染色体分离 → 等位基因A和a分别进入不同配子 → 两种配子比例1:1 → 测交后代Aa:aa = 1:1 → 表型比例1:1 → 验证分离定律
```

**得分要点:**
- 必须说明"1:1的理论基础是同源染色体分离"
- 需做X²检验确认偏离不显著
- 样本量需足够大(n≥100)以减少随机误差

#### 方法B: 自交法
**原理**: F2自交，F3表现特定的基因型比例

**实验步骤:**
1. F2中选取表型为显性的个体分别自交
2. 观察F3性状分离情况
3. F2显性个体中: 1/3纯合(AA)→F3不分离; 2/3杂合(Aa)→F3出现3:1分离

**得分要点:**
- 解释"显性个体包括AA和Aa两种基因型"
- 需说明"自交多代可使纯合子比例增加"

#### 方法C: 花粉鉴定法（快速检测）
**原理**: 某些基因的等位基因产物可在花粉粒中表达差异（如糯与非糯）

**实验步骤:**
1. 取F1杂合子花粉粒
2. 碘液染色（糯性淀粉遇碘呈红褐色，非糯呈蓝黑色）
3. 显微镜下计数两类花粉粒比例

**因果链:**
```
花粉粒(配子体)中单倍体基因表达 → 不同等位基因产物差异 → 染色表型不同 → 两种花粉比例1:1 → 验证减数分裂中等位基因分离
```

**得分要点:**
- 强调"花粉粒是单倍体，直接反映配子基因型"
- 适用于基因产物在花粉粒中可区分的性状

---

### 6.2 验证基因自由组合定律

#### 方法A: 测交法
**原理**: 双杂合子(AaBb)与双隐性(aabb)测交 → 后代应出现4种表型，比例1:1:1:1

**实验步骤:**
1. F1(AaBb) × 双隐性纯合(aabb)
2. 统计测交后代的表型种类与比例
3. 若4种表型比例接近1:1:1:1 → 支持自由组合

**因果链:**
```
F1(AaBb)减数分裂 → 非同源染色体自由组合 → 非等位基因自由组合 → 产生4种配子(AB,Ab,aB,ab)比例1:1:1:1 → 测交后代4种表型1:1:1:1 → 验证自由组合定律
```

**得分要点:**
- 强调"两对基因位于非同源染色体上"是前提
- 若偏离1:1:1:1 → 可能连锁，需进一步分析
- 需做X²检验（自由度=3）确认

#### 方法B: 自交法
**原理**: F1自交 → F2出现9:3:3:1（完全显性无互作）

**实验步骤:**
1. F1(AaBb)自交得到F2
2. 统计F2表型种类与比例
3. 若接近9:3:3:1 → 支持自由组合

**得分要点:**
- 需排除基因互作（9:7, 9:3:4等同样是两对基因但非自由组合的结果）
- 统计时需区分表型分类，不能用肉眼判断

---

### 6.3 判断基因位置（常染色体/性染色体）

#### 方法A: 正反交法
**原理**: 常染色体性状正反交结果一致；X连锁性状正反交结果不一致

**实验步骤:**
```
正交: 显性♀ × 隐性♂
反交: 隐性♀ × 显性♂
比较两组后代表型
```

**结果判断:**
| 正反交结果 | 判断 |
|-----------|------|
| 两组后代表型一致 | 常染色体遗传 |
| 正交后代全部显性，反交后代雌性显性雄性隐性 | X染色体隐性 |
| 正交后代全部显性，反交后代雌性显性雄性各半 | X染色体显性 |
| 正交反交结果均不同且有性别分化 | X染色体连锁 |

**因果链（以XR为例）:**
```
正交: X^AX^A × X^aY → 子代全部显性(♀X^AX^a, ♂X^AY)
反交: X^aX^a × X^AY → 子代♀全部显性(X^AX^a), ♂全部隐性(X^aY)
→ 正反交结果不同 → 排除常染色体 → 符合XR模式 → 基因在X染色体
```

**得分要点:**
- 必须同时做正反交，不能只做一组
- 若涉及多对基因，需逐一分析
- 细胞质遗传也呈正反交差异，但反交时后代更接近母本型

#### 方法B: 缺失定位法（河北进阶考点）
**原理**: 利用染色体片段缺失的个体进行互补测验，确定基因所在染色体区段

**实验步骤:**
1. 获得一系列已知缺失区间的缺失杂合体
2. 待测隐性突变个体与各缺失杂合体杂交
3. 若后代出现隐性表型 → 待测基因位于该缺失区间内

**因果链:**
```
隐性突变(a) × 缺失杂合(Del/+) → 若a在Del区间内 → 后代基因型: Del/a(仅一条有该基因且为突变型) → 表型为隐性 → 定位成功
```

---

### 6.4 判断显隐性关系

#### 方法A: 自交法（适用植物）
**步骤:**
1. 同一性状的两个相对性状纯合个体分别自交
2. 观察后代是否发生性状分离
3. 性状分离的亲本为显性杂合子（在后代中重新出现隐性）
4. 不分离的亲本可能为显性纯合或隐性纯合

#### 方法B: 杂交法（性状来源未知时）
**步骤:**
1. 将两种表型个体杂交
2. 观察F1全部个体的表型
3. F1表现出的性状 → 显性（或至少F1全部一致）
4. 若F1出现性状分化 → 亲本中有一个为杂合子，显性的判断需F2验证

**因果链（杂交法）:**
```
P: 表型I × 表型II
若F1全部为表型I → 表型I为显性(不完全显性时可能呈中间型)
I基因型: AA, II基因型: aa → F1: Aa → 全部表型I → I为完全显性
若F1出现两种表型 → 亲本之一为杂合子，需F2进一步验证
```

**得分要点:**
- 写清楚"若F1全部表现某一性状 → 该性状为显性"的逻辑
- 需排除不完全显性（此时F1呈中间型 = 共显性或不完全显性）

---

### 6.5 验证基因连锁

**原理**: 两对基因若连锁 → 测交后代表型比例偏离1:1:1:1 → 亲本型多，重组型少

**实验步骤:**
1. F1(AaBb) × 双隐性(aabb)测交
2. 统计4种表型数量
3. 计算重组率r = 重组型数量/总数量
4. 若四种表型数量显著偏离1:1:1:1且r<50% → 基因连锁
5. 进一步计算遗传距离 = r × 100(cM)

**结果判读（测交法）:**
```
F1(AaBb) × aabb → F2表型:
AB/ab型: 亲本型p1
Ab/ab型: 重组型r1
aB/ab型: 重组型r2
ab/ab型: 亲本型p2

若 p1+p2 >> r1+r2 → 基因连锁（亲本型远多于重组型）
重组率 = (r1+r2) / (p1+p2+r1+r2) × 100%
```

**因果链（连锁机制）:**
```
两基因位于同一染色体 → 减数分裂时同源染色体非姐妹染色单体间可能发生交叉互换 → 互换发生位置决定是否重组 → 紧密连锁的基因间互换概率低 → 重组型配子少 → 测交后代中亲本型远多于重组型 → 证实连锁
```

**得分要点:**
- 强调"连锁≠不重组"，连锁基因也有交换可能
- 连锁交换值≤50%（>50%视为自由组合）
- 连锁群: 位于同一染色体上的全部基因构成了连锁群
- 完全连锁: r=0（如雄果蝇、雌蚕）
- 不完全连锁: 0<r<50%

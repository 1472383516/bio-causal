# 高中生物概念因果词典

> 适用于高考生物因果推导引擎的概念参考文件  
> 共收录 200+ 核心概念，覆盖五大模块：分子与细胞 / 遗传与进化 / 稳态与调节 / 生态 / 生物工程  
> 格式：定义 → 因果机制 → 上位概念 → 下位实例 → 河北陷阱 → 关联概念

## 第一模块：分子与细胞

### 细胞
**定义**: 细胞是生物体结构和功能的基本单位，是最基本的生命系统层次。除病毒外，所有生物都由细胞构成。
**因果机制**: 细胞通过细胞膜与外界进行物质交换（输入营养、排出废物）→ 细胞质中进行代谢反应（酶催化）→ 细胞核储存遗传信息并调控基因表达 → 细胞分裂实现生长和繁殖。细胞的结构完整性是其功能的基础，结构破坏必然导致功能丧失。
**上位概念**: 生命系统的结构层次（细胞→组织→器官→系统→个体→种群→群落→生态系统→生物圈）
**下位实例**: 神经细胞（传导兴奋）、红细胞（运输O₂）、叶肉细胞（光合作用）、根毛细胞（吸收水分）
**河北陷阱**: 病毒没有细胞结构，所以病毒不属于生命系统的结构层次中的"细胞"层次，但病毒属于生物；单细胞生物既是细胞层次也是个体层次
**关联概念**: 细胞学说（施莱登和施旺提出，一切动植物都由细胞构成，细胞来自细胞的分裂），病毒（无细胞结构，必须寄生在活细胞内）

### 原核细胞
**定义**: 没有以核膜为界限的细胞核的细胞，遗传物质集中在拟核区域。主要包括细菌、蓝藻（蓝细菌）、放线菌、支原体、衣原体、立克次氏体。
**因果机制**: 原核细胞没有核膜→DNA集中在拟核区→转录和翻译在细胞质中几乎同时同地进行→繁殖速度快。原核细胞只有核糖体一种细胞器→蛋白质合成能力有限但有保证。多数原核细胞有细胞壁（主要成分为肽聚糖）→维持细胞形态和抗渗透压。
**上位概念**: 细胞分类（原核细胞 vs 真核细胞）→ 生物分类（原核生物 vs 真核生物）
**下位实例**: 大肠杆菌（肠道共生菌）、蓝藻（含叶绿素和藻蓝素，能进行光合作用）、乳酸菌（无氧呼吸产生乳酸）、硝化细菌（化能合成作用）
**河北陷阱**: 蓝藻是原核生物，能光合作用但没有叶绿体（只有光合片层）；原核生物不一定都有细胞壁（如支原体最小，无细胞壁）；"菌"字不一定指原核（酵母菌是真核，霉菌是真核）
**关联概念**: 真核细胞（进化上由原核细胞经内共生产生），细胞器（原核细胞只有核糖体），拟核（无核膜包裹的DNA集中区域），质粒（原核细胞中的小型环状DNA）

### 真核细胞
**定义**: 具有以核膜为界限的细胞核的细胞，细胞核内有染色体。包括动物细胞、植物细胞、真菌细胞、原生生物细胞等。
**因果机制**: 真核细胞的核膜将遗传信息与翻译在空间上分隔→转录在核内，翻译在细胞质→基因表达调控更精细→能分化形成多细胞生物的各种组织。真核细胞拥有丰富的膜性细胞器（内质网、高尔基体等）→生物膜系统将细胞内划分成不同功能区域→代谢效率大幅提高。
**上位概念**: 细胞分类 → 真核生物（原生生物界、真菌界、植物界、动物界）
**下位实例**: 动物细胞（无细胞壁、无液泡或小液泡）、植物细胞（有细胞壁、有大液泡、有叶绿体）、酵母菌（单细胞真菌）、变形虫（单细胞原生动物）
**河北陷阱**: 不是所有真核细胞都有细胞核（筛管细胞、成熟红细胞无核）；植物细胞特有结构：细胞壁、液泡、叶绿体（三者不一定同时存在，根细胞无叶绿体）
**关联概念**: 原核细胞（进化起源关系），生物膜系统（核膜+细胞器膜+细胞膜），细胞器（真核细胞有多种膜性细胞器和非膜性细胞器），内共生学说（线粒体和叶绿体起源于被真核祖先吞噬的原核生物）

### 细胞膜
**定义**: 包围在细胞外表面的一层薄膜，主要由磷脂双分子层和蛋白质组成，基本骨架为磷脂双分子层，具有流动性。
**因果机制**: 磷脂双分子层形成疏水屏障→阻止大分子和离子自由通过→蛋白质（载体蛋白、通道蛋白）介导选择性转运→实现选择透过性。糖蛋白分布于细胞膜外侧→参与细胞识别→免疫系统识别自我与非我。膜的流动性（磷脂和蛋白质可运动）→膜融合（胞吞胞吐）、膜修复。
**上位概念**: 生物膜系统（细胞膜是生物膜的组成部分，与其他生物膜结构和功能相似）
**下位实例**: 红细胞的细胞膜（分离膜蛋白经典材料）、神经细胞膜（含大量Na⁺通道和K⁺通道，参与动作电位）、小肠上皮细胞膜（面向肠腔面有微绒毛增大吸收面积）
**河北陷阱**: 细胞膜的功能特点是"选择透过性"，结构特点是"一定的流动性"；细胞膜的外侧有糖蛋白，内侧没有——可以用来判断膜的内外方向；不是所有物质都需要蛋白质参与跨膜运输（如自由扩散）
**关联概念**: 流动镶嵌模型（辛格和尼科尔森提出），选择透过性，胞吞/胞吐（依赖膜流动性），生物膜系统

### 细胞质
**定义**: 细胞膜以内、细胞核以外的部分，包括细胞质基质和悬浮于其中的细胞器。细胞质基质是胶状物质，含大量水和多种溶解物质。
**因果机制**: 细胞质基质提供稳定的离子环境和pH→保证酶促反应正常进行→代谢中间产物在基质中扩散→连接各细胞器的代谢。细胞骨架分布在细胞质中→维持细胞形态、锚定细胞器、参与胞内运输。细胞质流动（原生质环流）→促进物质分布和交换。
**上位概念**: 细胞结构（细胞膜+细胞质+细胞核）→ 细胞功能实现的基础
**下位实例**: 洋葱鳞片叶内表皮细胞的细胞质环流（观察细胞质流动的经典材料）、分泌细胞（高尔基体和内质网发达）、肌细胞（富含线粒体，细胞质中有大量肌原纤维）
**河北陷阱**: "细胞质"和"细胞质基质"不是同一概念，细胞质包括细胞器，细胞质基质不包括细胞器；细胞质基质是细胞代谢的主要场所（不是细胞核！）
**关联概念**: 细胞质基质（代谢主要场所），细胞器（悬浮在细胞质基质中），细胞骨架（分布在细胞质中），原生质（细胞膜+细胞质+细胞核=原生质，去掉细胞壁后的植物细胞=原生质体）

### 细胞核
**定义**: 真核细胞内最大的由膜包围的结构，是遗传信息库，是细胞代谢和遗传的控制中心。由核膜、核仁、染色质/染色体、核基质组成。
**因果机制**: 核膜上有核孔→允许mRNA、蛋白质等大分子选择性进出→实现核-质间物质运输和信息交流。染色质（分裂间期）→螺旋化→染色体（分裂期）→携带全部遗传信息→通过有丝分裂/减数分裂传递给子代。核仁与rRNA的合成和核糖体的形成有关→核糖体在细胞质中合成蛋白质。DNA→转录→mRNA→出核→翻译→蛋白质→执行细胞功能。
**上位概念**: 真核细胞结构 → 遗传控制中心（区别于原核细胞的拟核）
**下位实例**: 骨骼肌细胞多核（合胞体）、筛管细胞和成熟红细胞无核、肝细胞双核较多见、神经细胞核大而明显
**河北陷阱**: 细胞核不是细胞代谢的主要场所（代谢主要场所在细胞质基质），细胞核是代谢的控制中心；核膜是双层膜，每层都是磷脂双分子层，所以共4层磷脂分子层；核孔是有选择性的，不是任何物质都能自由通过；染色体和染色质是同种物质在不同时期的两种形态
**关联概念**: 核膜（双层膜，属于生物膜系统），核孔（大分子选择性运输通道），核仁（与rRNA和核糖体形成有关），染色质/染色体（遗传物质载体），转录（在细胞核中进行），拟核（原核细胞的对应结构）

### 线粒体
**定义**: 进行有氧呼吸的主要场所，具有双层膜结构，内膜向内折叠形成嵴。含有自身的DNA、RNA和核糖体，是一种半自主性细胞器。
**因果机制**: 外膜通透性较高→小分子自由通过→内膜含有呼吸链酶复合体和ATP合酶→糖酵解产物丙酮酸进入线粒体基质→经柠檬酸循环产生NADH和FADH₂→呼吸链电子传递→建立H⁺梯度→驱动ATP合酶合成大量ATP。嵴的存在增大了内膜面积→容纳更多呼吸链酶→提高ATP产出效率。
**上位概念**: 细胞器 → 膜性细胞器 → 半自主性细胞器（自身有DNA和核糖体）
**下位实例**: 肌细胞（耗能高，线粒体丰富）、精子颈部（线粒体围绕鞭毛基部提供能量）、肝细胞（代谢活跃，线粒体多）、棕色脂肪细胞（线粒体内膜含UCP1解偶联蛋白，产热）
**河北陷阱**: 线粒体是"有氧呼吸的主要场所"而非"唯一场所"（有氧呼吸第一阶段糖酵解在细胞质基质）；无氧呼吸不需要线粒体，全过程在细胞质基质；线粒体内膜面积大于外膜（因嵴的存在）；蓝藻等原核生物没有线粒体但也能进行有氧呼吸（有与呼吸相关的酶分布在细胞膜和细胞质）
**关联概念**: 有氧呼吸（主要场所在线粒体），叶绿体（同为双层膜半自主性细胞器），内共生学说（线粒体起源于被吞噬的好氧细菌），ATP（线粒体是ATP的主要来源），嵴（增大内膜面积），细胞质基质（糖酵解的场所）

### 叶绿体
**定义**: 进行光合作用的细胞器，具有双层膜结构，内部有基粒（由类囊体垛叠而成）和基质。含有自身的DNA、RNA和核糖体，是半自主性细胞器。
**因果机制**: 类囊体膜上含光合色素（叶绿素a、b和类胡萝卜素）和光合作用相关酶→光能→类囊体膜上进行光反应（水的光解和光合磷酸化）→产生ATP和NADPH→基质中进行暗反应（卡尔文循环）→CO₂固定和C₃还原→生成糖类。基粒垛叠增大了膜面积→容纳更多色素和酶→提高光合效率。
**上位概念**: 细胞器 → 膜性细胞器 → 质体 → 半自主性细胞器
**下位实例**: 叶肉细胞（含大量叶绿体）、保卫细胞（含叶绿体，参与气孔开闭）、C₄植物维管束鞘细胞（含叶绿体但基粒不发达）、CAM植物（夜间储存CO₂）
**河北陷阱**: 叶绿体不是光合作用"唯一"场所（蓝藻无叶绿体，在光合片层上进行光合作用）；叶绿体中的色素只分布在类囊体膜上，基质中没有光合色素；并非所有植物细胞都有叶绿体（根细胞没有）；液泡中的色素不是光合色素（花青素等，决定花和果实的颜色）
**关联概念**: 光合作用（光反应在类囊体膜上进行，暗反应在基质中进行），线粒体（同为双层膜半自主性细胞器），内共生学说（叶绿体起源于被吞噬的蓝藻），光合色素（叶绿素a、b、类胡萝卜素），质体（白色体、有色体、叶绿体可相互转化）

### 内质网
**定义**: 由单层膜围成的相互连通的扁平囊状和管状结构，分为粗面内质网（表面附着核糖体）和滑面内质网（表面无核糖体）。
**因果机制**: 粗面内质网上核糖体合成的蛋白质进入内质网腔→蛋白质在腔内折叠、修饰（如糖基化）→通过囊泡运输至高尔基体→进一步加工和分选。滑面内质网参与脂质合成（磷脂、固醇类）→新合成的脂质掺入内质网膜→膜面积增大→膜流维持。内质网还与Ca²⁺储存（肌细胞中为肌质网）和解毒功能（肝细胞中）有关。
**上位概念**: 细胞器 → 膜性细胞器 → 生物膜系统（与核膜、高尔基体膜、细胞膜在结构和功能上有联系）
**下位实例**: 浆细胞（大量粗面内质网，合成抗体）、肝细胞（滑面内质网发达，参与解毒）、肌细胞（肌质网是特化的滑面内质网，储存和释放Ca²⁺）
**河北陷阱**: 内质网是单层膜结构，不是双层膜；内质网与核膜相连（不是与细胞膜直接相连）；粗面和滑面内质网在功能上不同，但它们是一个连续的膜系统（可以相互转化）
**关联概念**: 核糖体（附着在内质网上），高尔基体（接受来自内质网的囊泡，进一步加工蛋白质），膜流（内质网→高尔基体→细胞膜的膜流动），分泌蛋白（在粗面内质网上合成并进入内质网腔），脂质合成（滑面内质网的主要功能）

### 高尔基体
**定义**: 由单层膜围成的一叠扁平囊泡及其周围的小囊泡组成的细胞器，在动植物细胞中均有，但功能有所不同。
**因果机制**: 来自内质网的囊泡与高尔基体顺面融合→蛋白质在高尔基体中经过糖基化修饰→在高尔基体反面形成分泌囊泡或溶酶体→分泌到细胞外或在细胞内发挥作用。在植物细胞中，高尔基体参与细胞壁的形成（分泌果胶等多糖类物质→构成细胞壁纤维素的合成和分泌）。
**上位概念**: 细胞器 → 膜性细胞器 → 生物膜系统（连接内质网和细胞膜）
**下位实例**: 分泌细胞（高尔基体发达，如胰腺细胞分泌消化酶）、植物分裂细胞（高尔基体参与形成细胞板→新细胞壁）
**河北陷阱**: 高尔基体在动植物细胞中都有（不是植物特有）；植物细胞高尔基体与细胞壁形成有关（动物细胞高尔基体与分泌物形成有关）；高尔基体和内质网都有加工蛋白质的功能，区别在于内质网侧重折叠和初步糖基化，高尔基体侧重进一步加工和分选
**关联概念**: 内质网→高尔基体→细胞膜（分泌途径），囊泡运输，分泌蛋白（经高尔基体分泌），溶酶体（在动物细胞中由高尔基体出芽形成），细胞板（植物细胞分裂时，高尔基体小泡聚集形成细胞板）

### 溶酶体
**定义**: 由单层膜包围的球形细胞器，内含多种酸性水解酶，是细胞内的"消化车间"。主要分布在动物细胞中。
**因果机制**: 溶酶体内的酶在酸性环境（pH约5）中活性最高→溶酶体膜上的H⁺泵维持腔内酸性环境→溶酶体与吞噬泡或自噬泡融合→水解酶分解其中的物质→产物进入细胞质基质被重新利用。溶酶体也参与细胞的程序性死亡（凋亡）→溶酶体膜破裂→水解酶释放到细胞质中→细胞自溶。
**上位概念**: 细胞器 → 膜性细胞器（由高尔基体出芽形成）→ 细胞内消化系统
**下位实例**: 白细胞中的溶酶体（消化吞噬的病菌）、蝌蚪尾部的退化（溶酶体参与细胞自溶）、内质网应激时自噬增加（溶酶体分解受损内质网）
**河北陷阱**: 溶酶体主要存在于动物细胞中（植物细胞中较少见或功能由液泡替代）；溶酶体中的酶是酸性水解酶（最适pH为酸性），与细胞质基质（pH约7.2）不同，这保证了溶酶体破裂后酶不会立刻损伤细胞（因pH不适）；"溶酶体来自高尔基体"的说法不完全准确——初级溶酶体来自高尔基体
**关联概念**: 高尔基体（溶酶体的来源），胞吞（吞噬体与溶酶体融合），自噬（受损细胞器被溶酶体分解），细胞凋亡（溶酶体参与），酶（溶酶体中含有多种水解酶）

### 核糖体
**定义**: 由rRNA和蛋白质组成的非膜性细胞器，是蛋白质合成的场所。由大亚基和小亚基组成，存在于所有细胞（包括原核和真核细胞）中。
**因果机制**: mRNA从细胞核通过核孔进入细胞质→与核糖体小亚基结合→tRNA携带氨基酸与mRNA上的密码子配对→大亚基催化肽键形成→多肽链延伸→核糖体沿着mRNA移动（移位）→直至遇到终止密码子→多肽链释放。多个核糖体可以同时附着在同一条mRNA上形成多聚核糖体→大大提高翻译效率。
**上位概念**: 细胞器 → 非膜性细胞器 → 原核和真核细胞的共性结构（支持共同起源学说）
**下位实例**: 网织红细胞（无核但含大量核糖体，继续合成血红蛋白）、浆细胞（粗面内质网上有大量核糖体合成抗体）、细菌（核糖体为70S，与真核细胞的80S不同）
**河北陷阱**: 核糖体无膜结构，不属于生物膜系统；核糖体是所有细胞共有的细胞器（原核、真核都有），这支持生物有共同起源；原核细胞的核糖体（70S）与真核细胞细胞质中的核糖体（80S）不同，但线粒体和叶绿体中的核糖体与细菌核糖体相似（支持内共生学说）；核糖体在细胞中的存在形式有游离型和附着型两种
**关联概念**: 蛋白质合成（翻译），mRNA（携带遗传信息到核糖体），tRNA（搬运氨基酸），rRNA（核糖体的组成成分），粗面内质网（附有核糖体），多聚核糖体（提高翻译效率）

### 中心体
**定义**: 由两个相互垂直的中心粒及其周围的中心粒周围物质组成的非膜性细胞器，存在于动物细胞和低等植物细胞中，与细胞分裂有关。
**因果机制**: 分裂间期中心体复制→分裂前期两组中心体分别移向细胞两极→发出星射线形成纺锤体→纺锤丝附着在染色体的着丝粒上→牵引染色体分离→有丝分裂后期/减数分裂后期染色体移向两极。没有中心体，纺锤体无法正常形成→染色体不能正常分离→子细胞染色体数目异常。
**上位概念**: 细胞器 → 非膜性细胞器 → 微管组织中心（组织微管形成纺锤体）
**下位实例**: 动物细胞有丝分裂（中心体参与纺锤体形成）、低等植物（如衣藻）有丝分裂、纤毛和鞭毛的基部（基体，结构与中心粒相同）
**河北陷阱**: 高等植物细胞没有中心体，也能分裂（由细胞两极发出纺锤丝形成纺锤体）；中心体复制发生在分裂间期（与DNA复制同步）；不仅参与有丝分裂，也参与减数分裂
**关联概念**: 有丝分裂（中心体移向两极→纺锤体形成→染色体分离），减数分裂，微管（纺锤丝由微管蛋白组成），星射线（由中心体发出的放射状微管）

### 液泡
**定义**: 由单层膜（液泡膜）包围的囊状结构，内含细胞液（含糖类、无机盐、色素、有机酸等）。在植物细胞中，成熟细胞往往有一个大的中央液泡；动物细胞中液泡较小或不明显。
**因果机制**: 液泡内溶质浓度高→细胞液渗透压高于细胞外→水分通过渗透作用进入液泡→液泡吸水膨胀→对细胞壁产生膨压→维持植物细胞的紧张状态（挺立）。液泡储存各种代谢产物（如花青素决定花色、有机酸决定酸味）→参与植物防御和吸引传粉者。液泡还参与细胞内物质的分解和废物储存→类似动物细胞的溶酶体功能。
**上位概念**: 细胞器 → 膜性细胞器（单层膜）→ 植物细胞内环境调节的关键结构
**下位实例**: 洋葱外表皮细胞（紫色大液泡便于观察质壁分离）、甘蔗茎细胞（液泡储存大量蔗糖）、花瓣细胞（液泡含花青素决定颜色）、保卫细胞（液泡渗透压变化驱动气孔开闭）
**河北陷阱**: 植物细胞不一定都有大液泡（分生组织细胞液泡小且多，成熟细胞才有中央大液泡）；液泡中的色素（花青素等）不是光合色素，分布在液泡液中，决定花、果实等器官的颜色；液泡膜是单层膜，选择透过性
**关联概念**: 渗透作用（液泡渗透压驱动水分进出细胞），质壁分离（液泡失水导致原生质体缩小），花青素（分布在液泡中，颜色随pH改变），细胞壁（液泡的膨压使植物细胞紧张），溶酶体（植物细胞的液泡有类似溶酶体的消化功能）

### 细胞壁
**定义**: 包围在细胞膜外的一层较坚硬的结构，主要由纤维素组成（植物细胞），或由肽聚糖组成（细菌细胞壁），或由几丁质组成（真菌细胞壁）。动植物细胞的主要区别之一：植物细胞有细胞壁，动物细胞无。
**因果机制**: 细胞壁提供机械支撑→维持细胞形态→保护细胞不受机械损伤和渗透压变化的影响。细胞壁上的纹孔和胞间连丝→使相邻细胞的原生质体相互连通→物质和信息可以交换。纤维素分子聚集成微纤丝→构成细胞壁的骨架→果胶等多糖填充其中→半纤维素交联→形成多孔但坚韧的结构。
**上位概念**: 细胞结构（与细胞膜共同构成细胞外侧边界）→ 植物细胞区别于动物细胞的特征之一
**下位实例**: 木质部导管细胞壁加厚并木质化（次生壁，形成导管）、韧皮部筛管（有细胞壁但无细胞核）、根毛（细胞壁向外凸起增大吸收面积）、细菌细胞壁（肽聚糖，革兰氏阳性菌壁厚，阴性菌壁薄）
**河北陷阱**: 细胞壁是全透性的（不是选择透过性——选择透过性是细胞膜的功能特点）；去掉细胞壁的植物细胞称为"原生质体"；植物细胞壁的主要成分是纤维素和果胶，细菌细胞壁的主要成分是肽聚糖，真菌细胞壁的主要成分是几丁质；原核细胞大多数有细胞壁（支原体例外）
**关联概念**: 细胞膜（紧贴在细胞壁内侧，具选择透过性），纤维素（组成细胞壁的多糖），原生质体（去除细胞壁后的植物细胞），质壁分离（原生质体与细胞壁分离），胞间连丝（穿过细胞壁连接相邻细胞的通道）

### 细胞骨架
**定义**: 由微管、微丝（肌动蛋白丝）和中间纤维组成的蛋白质纤维网络体系，分布在细胞质中，维持细胞的形态并参与细胞的各种运动。
**因果机制**: 微管维持细胞形态→作为胞内运输的轨道→驱动蛋白和动力蛋白沿微管运输囊泡和细胞器→形成纺锤体牵拉染色体。微丝（肌动蛋白丝）在细胞膜下方形成网络→支撑细胞膜→参与细胞运动（伪足形成、胞质分裂）。中间纤维提供机械强度→锚定细胞器和核→抵抗机械应力。
**上位概念**: 细胞质结构 → 细胞运动与维持形态的系统 → 真核细胞特有的结构
**下位实例**: 神经细胞轴突中的微管（物质运输轨道）、肌细胞中的微丝（与肌球蛋白相互作用产生收缩）、上皮细胞的中间纤维（角蛋白，提供强度）、分裂细胞的纺锤体（微管组成）
**河北陷阱**: 细胞骨架不是"骨骼"那样坚硬——它是动态的蛋白质纤维，不断组装和去组装；细胞骨架在真核细胞中存在，原核细胞没有；微管和微丝都是动态结构（可以聚合和解聚），这一点与维持细胞形态看似矛盾实则统一（动态平衡）
**关联概念**: 微管（组成纺锤体、纤毛、鞭毛），微丝（参与胞质分裂和变形运动），中间纤维（提供机械强度），纺锤体（微管组成），中心体（微管组织中心），马达蛋白（沿微管运输物质）

### 生物膜系统
**定义**: 真核细胞中由细胞膜、核膜以及各种膜性细胞器（内质网、高尔基体、线粒体、叶绿体、溶酶体、液泡等）的膜共同构成的在结构和功能上有联系的统一整体。
**因果机制**: 各生物膜的化学组成基本相似（主要是磷脂和蛋白质）→膜之间可以通过囊泡间接相连（内质网→高尔基体→细胞膜）或直接相连（内质网与核膜）→实现物质的定向运输和膜的更新。生物膜将细胞分隔成不同区域→各种生化反应在不同膜结构中进行→互不干扰→提高代谢效率。膜面积大→为酶提供附着位点→利于生化反应有序进行。
**上位概念**: 真核细胞特有的结构系统 → 细胞区域化（区室化）的基础
**下位实例**: 分泌蛋白的合成、加工和分泌途径（核糖体→内质网→高尔基体→细胞膜）、胞吞和胞吐过程中的膜融合和膜回收、内共生学说中的膜演化
**河北陷阱**: 生物膜系统是"真核细胞"特有的，原核细胞只有细胞膜一种生物膜，没有内膜系统；核糖体、中心体不是膜性细胞器，不属于生物膜系统；生物膜系统强调"结构和功能上有联系"，不是简单的膜的总和
**关联概念**: 流动镶嵌模型（生物膜的结构模型），选择透过性（生物膜的功能特点），膜流（生物膜之间的动态联系），区室化（生物膜分隔出的功能区域），分泌蛋白（合成和运输依赖生物膜系统）

### 流动镶嵌模型
**定义**: 辛格（Singer）和尼科尔森（Nicolson）于1972年提出的生物膜结构模型。基本内容：磷脂双分子层构成膜的基本骨架，蛋白质分子镶嵌在磷脂双分子层中（或贯穿、或附着），磷脂分子和大多数蛋白质分子都可以运动，膜具有流动性。
**因果机制**: 磷脂分子的疏水尾部朝向内部、亲水头部朝向外部→形成稳定的双分子层→形成疏水屏障→阻止离子和大分子自由通过。蛋白质嵌入或贯穿磷脂双分子层→形成通道或载体→物质选择性跨膜运输。膜蛋白和磷脂均可运动→膜具有一定的流动性→膜的融合（细胞融合、胞吞胞吐、囊泡运输）成为可能。膜的不对称性（内外两侧的蛋白质和糖类分布不同）→膜的功能具有方向性。
**上位概念**: 生物膜的结构模型（取代了之前的"三明治模型"）→ 生物膜结构和功能的认识基础
**下位实例**: 荧光抗体标记人-鼠细胞融合实验（证明膜蛋白可以运动）、冰冻蚀刻电镜技术（展示膜的内部结构）、变形虫的变形运动（依赖膜的流动性）
**河北陷阱**: "流动"中的主体是"磷脂"和"蛋白质"都流动；"镶嵌"指蛋白质在磷脂双分子层中有多种存在方式（贯穿、嵌入、附着）；膜的流动性是结构特点，选择透过性是功能特点——二者不混淆；细胞膜外侧有糖蛋白和糖脂（形成糖萼），内侧没有
**关联概念**: 细胞膜（流动镶嵌模型是其结构理论），选择透过性（功能特点，由结构决定），磷脂双分子层（基本骨架），膜蛋白（功能的主要执行者），糖蛋白（细胞识别）

### 选择透过性
**定义**: 细胞膜的功能特性——允许某些物质通过而限制或阻止其他物质通过的能力。这是活细胞的重要特征，死细胞丧失选择透过性。
**因果机制**: 磷脂双分子层内部疏水→非极性小分子（如O₂、CO₂、乙醇）可通过自由扩散通过→极性分子和离子被疏水区域排斥。膜蛋白形成通道或载体→离子（Na⁺、K⁺、Ca²⁺）可通过通道蛋白顺浓度梯度转运→或通过载体蛋白顺/逆浓度梯度转运（主动运输）。选择透过性使细胞能维持内环境与外界环境的差异→保证正常代谢。
**上位概念**: 细胞膜的功能特点（结构特点是流动性）→ 活细胞的特征（死细胞丧失选择透过性）
**下位实例**: 红细胞的渗透实验（在不同浓度NaCl溶液中形态改变）、植物细胞的质壁分离与复原（活细胞的选择透过性使水分可进出而蔗糖不能自由通过）、肾小管的重吸收（上皮细胞的选择性转运）
**河北陷阱**: 选择透过性是"功能特点"，流动镶嵌模型是"结构模型"，两者不能混淆；死细胞失去选择透过性→变成"全透性"→可以用台盼蓝染色鉴别活细胞（活细胞拒染，死细胞染成蓝色）；选择透过性与"半透性"的区别：半透性只与分子大小有关（物理性质），选择透过性还与生物活性有关
**关联概念**: 流动镶嵌模型（结构基础），自由扩散（依赖选择透过性），协助扩散（依赖载体蛋白或通道蛋白），主动运输（由载体蛋白介导的逆浓度转运），质壁分离（选择透过性的体现）

### 自由扩散
**定义**: 物质从高浓度一侧通过细胞膜运输到低浓度一侧，不需要载体蛋白的协助，也不消耗能量（ATP）。又称简单扩散。
**因果机制**: 分子在溶液中的热运动→物质从高浓度区域向低浓度区域净移动→浓度差越大→扩散速率越快→最终达到动态平衡（两侧浓度相等时净扩散为零，但分子仍在运动）。磷脂双分子层的疏水内部允许非极性小分子通过→O₂、CO₂等气体分子直接穿过膜→不消耗能量→属于被动运输。
**上位概念**: 被动运输（不需要能量）→ 物质跨膜运输方式 → 仅依靠浓度梯度的运输
**下位实例**: O₂从肺泡进入血液（高浓度→低浓度）、CO₂从组织细胞排出到血液、甘油被小肠上皮细胞吸收、乙醇进入细胞、水分子通过渗透作用（也属于自由扩散的一种特例，或通过水通道蛋白的协助扩散）
**河北陷阱**: 自由扩散不需要"载体蛋白"，但不等于不需要"通道蛋白"的运输都是自由扩散（协助扩散中的通道蛋白介导也属于协助扩散）；"渗透作用"是水分子的自由扩散——有争议，有些教材认为水可通过自由扩散和协助扩散（水通道蛋白/水孔蛋白）两种方式；自由扩散速率与浓度差成正比，不受载体蛋白数量的限制
**关联概念**: 协助扩散（也需要浓度差但不需能量，需要转运蛋白），主动运输（需要能量，可逆浓度梯度），选择透过性（决定哪些物质可自由扩散），被动运输（包括自由扩散和协助扩散），渗透作用（水的被动运输）

### 协助扩散
**定义**: 物质从高浓度一侧通过细胞膜运输到低浓度一侧，需要细胞膜上载体蛋白或通道蛋白的协助，但不消耗能量（ATP）。属于被动运输的一种。
**因果机制**: 载体蛋白与被运输物质特异性结合→载体蛋白构象改变→物质被转运到膜另一侧→载体蛋白恢复原构象。通道蛋白形成贯穿膜的水性孔道→特定离子/分子通过孔道→通道的开放和关闭可受多种因素调控（如电压门控、配体门控）。协助扩散的速率受转运蛋白的数量限制→当所有转运蛋白均饱和后→速率不再随浓度差增大而增大。
**上位概念**: 被动运输（不消耗能量）→ 物质跨膜运输方式 → 转运蛋白介导的被动运输
**下位实例**: 葡萄糖进入红细胞（载体蛋白型GLUT1协助扩散）、神经细胞Na⁺内流（电压门控Na⁺通道）、K⁺外流（电压门控K⁺通道）、水分子通过水通道蛋白/水孔蛋白（水通道蛋白介导的协助扩散）
**河北陷阱**: 协助扩散属于"被动运输"（不消耗ATP），常与主动运输混淆——关键区别是能量来源和浓度方向（顺浓度vs逆浓度）；葡萄糖进入小肠上皮细胞是"主动运输"（Na⁺-葡萄糖同向转运），而进入红细胞是"协助扩散"——同一物质在不同细胞中运输方式可能不同；协助扩散达到饱和后速率不再增加（区别于自由扩散）
**关联概念**: 自由扩散（不需转运蛋白的被动运输），主动运输（需要能量，可逆浓度梯度），载体蛋白（协助扩散和主动运输共用），通道蛋白（离子通道，协助扩散），被动运输（不耗能的跨膜运输方式总和）

### 主动运输
**定义**: 物质从低浓度一侧通过细胞膜运输到高浓度一侧，需要载体蛋白的协助且消耗能量（ATP或离子梯度提供的能量）。可以逆浓度梯度运输物质。
**因果机制**: 载体蛋白与低浓度侧的特异性物质结合→消耗能量使载体蛋白构象改变→物质被释放到高浓度侧→载体蛋白恢复原构象。能量来源分为两种：①原发性主动运输——直接消耗ATP（如Na⁺-K⁺泵）；②继发性主动运输——利用原发性主动运输建立的离子梯度作为驱动力（如Na⁺-葡萄糖同向转运，Na⁺顺梯度进入细胞的同时葡萄糖逆梯度进入）。主动运输维持细胞内外的离子浓度差→形成静息电位→保证神经传导等生命活动。
**上位概念**: 物质跨膜运输方式（区别于被动运输）→ 逆浓度梯度运输（需要外源能量）
**下位实例**: Na⁺-K⁺泵（每消耗1个ATP泵出3个Na⁺、泵入2个K⁺）、小肠上皮细胞吸收葡萄糖和氨基酸（Na⁺-葡萄糖/氨基酸同向转运）、甲状腺滤泡细胞富集I⁻（Na⁺-I⁻同向转运）、植物根细胞从低浓度的土壤溶液中吸收矿质元素
**河北陷阱**: 主动运输一定需要"载体蛋白"，但不是所有需要载体蛋白的运输都是主动运输（协助扩散也需要载体蛋白/通道蛋白但不耗能）；主动运输可以逆浓度梯度（低→高），也可以顺浓度梯度（只要有载体蛋白和耗能的情况下的顺浓度运输也是主动运输——但这种情况下以逆浓度为主要特征）；胞吞胞吐不是主动运输（虽然也消耗能量），它们是大分子/颗粒的跨膜方式
**关联概念**: Na⁺-K⁺泵（原发性主动运输的经典实例），协助扩散（需要转运蛋白但不耗能），被动运输（顺浓度梯度不耗能），静息电位（由Na⁺-K⁺泵和K⁺泄漏共同维持），离子梯度（继发性主动运输的能量来源）

### 胞吞
**定义**: 细胞膜向内凹陷，将外界的大分子/颗粒包裹起来，形成囊泡进入细胞内的过程。需要消耗ATP，依赖膜的流动性。
**因果机制**: 外界物质与细胞膜上的受体结合→触发细胞膜局部凹陷→细胞膜包裹物质形成吞噬泡或吞饮泡→囊泡脱离细胞膜进入细胞质→吞噬泡与溶酶体融合→溶酶体中的水解酶分解囊泡内物质→产物释放到细胞质中被利用。胞吞过程中，细胞膜面积减小（部分膜形成囊泡进入细胞内）→通过胞吐回收膜成分以维持膜平衡。
**上位概念**: 物质跨膜运输方式（大分子/颗粒的运输，不同于跨膜运输）→ 耗能且依赖膜流动性的运输方式
**下位实例**: 白细胞吞噬病菌（吞噬作用）、小肠上皮细胞吸收脂蛋白颗粒、细胞摄取低密度脂蛋白（LDL，受体介导的胞吞）、变形虫摄取食物颗粒
**河北陷阱**: 胞吞不是"主动运输"——主动运输指通过载体蛋白的跨膜运输，胞吞是膜包裹形成囊泡的过程；胞吞消耗ATP但不需要载体蛋白；胞吞的"是跨膜运输吗"——不是跨膜运输（物质没有穿过膜），而是膜包裹物质后囊泡脱离进入细胞
**关联概念**: 胞吐（胞吞的逆过程），溶酶体（与吞噬体融合进行消化），细胞膜流动性（胞吞的结构基础），受体（受体介导的胞吞），高尔基体（提供膜成分供胞吞补充膜面积）

### 胞吐
**定义**: 细胞内的囊泡与细胞膜融合，将囊泡内的物质释放到细胞外的过程。需要消耗ATP，依赖膜的流动性。
**因果机制**: 分泌蛋白（如激素、消化酶、抗体）在粗面内质网合成→经高尔基体加工和分选→以分泌囊泡形式运输到细胞膜→囊泡膜与细胞膜融合→囊泡内容物被释放到胞外→囊泡膜整合到细胞膜中→细胞膜面积增加。胞吐分为组成性胞吐（持续不断地进行）和调节性胞吐（在信号刺激下才进行，如神经递质释放）。
**上位概念**: 物质跨膜运输方式（大分子/颗粒的排出方式）→ 耗能且依赖膜流动性的排出 → 胞吞的逆过程
**下位实例**: 神经递质释放（突触小泡与突触前膜融合释放神经递质）、消化酶的分泌（胰腺细胞分泌胰蛋白酶等）、激素分泌（胰岛B细胞分泌胰岛素）、抗体的分泌（浆细胞分泌抗体）、细胞壁物质的分泌（植物细胞高尔基体小泡释放纤维素等多糖）
**河北陷阱**: 胞吐不是扩散（物质未穿过膜），不是主动运输（不经载体蛋白）；胞吐消耗能量（ATP）但不需载体蛋白；神经递质通过胞吐释放→在突触间隙中扩散→与突触后膜受体结合→不是跨膜进入突触后神经元
**关联概念**: 胞吞（胞吐的逆过程），分泌蛋白（经胞吐释放），高尔基体（形成分泌囊泡），神经递质（通过胞吐释放），细胞膜流动性（胞吐的结构基础），膜流（胞吞和胞吐参与膜的循环）

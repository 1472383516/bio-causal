# 变式追问生成模板库

> 供 bio-causal-reasoning Skill 在执行第四步时查阅。
> 基于因果链自动生成高质量变式追问。

---

## 一、五大变式类型详解

### 类型1：条件反转（Condition Reversal）

**定义**：改变因果链中某一环节的条件（移除/破坏/增强），问下游结果如何变化。

**生成模板**：
```
"若{必要条件X}被移除/抑制/增强，{下游现象Y}会发生什么变化？请说明原因。"
"在{特殊条件}下，{原本的因果链}还能成立吗？为什么？"
```

**高中生物典型示例**：
- "若类囊体薄膜被破坏，光反应还能进行吗？ATP和NADPH的产量如何变化？"
- "若用药物抑制纺锤体形成，细胞分裂会停留在哪个时期？"
- "若切除垂体，甲状腺激素的分泌如何变化？"
- "若将植物从光下移到暗处，RuBP和3-PGA的浓度如何变化？"

**考察层次**：因果关系理解（知道X→Y还不够，还要知道没有X会怎样）

---

### 类型2：环节跨越（Link Skipping）

**定义**：跳过中间环节，直接问因果链两端的关系，检验是否理解中间机制的约束。

**生成模板**：
```
"为什么{上游事件A}直接影响{下游事件Z}，而不需要经过{中间环节B}？这个说法对吗？"
"有人说'{错误跨越说法}'，请判断正误并说明原因。"
```

**高中生物典型示例**：
- "为什么说'光反应为暗反应提供ATP和NADPH，所以直接抑制暗反应，光反应也会受影响'？这个说法对吗？"
- "ATP可以穿过线粒体内膜直接进入细胞质吗？如果不能，那细胞质中的糖酵解如何获得ATP？"
- "DNA直接控制性状吗？为什么基因突变不一定导致性状改变？"

**考察层次**：中间机制细节的掌握程度

---

### 类型3：类比迁移（Analogy Transfer）

**定义**：将因果逻辑套用到结构相似但场景不同的生物学过程中。

**生成模板**：
```
"{过程A}的因果逻辑与{过程B}有何相似和不同？请列表对比。"
"如果把{过程A}中的{组件X}换成{过程B}中的{组件Y}，结果会怎样？"
```

**高中生物典型示例**：
- "光合作用中类囊体薄膜上的电子传递链，与线粒体内膜上的电子传递链，因果逻辑有何异同？"
- "激素调节和神经调节，在'信号→受体→效应'这个因果链上有什么本质区别？"
- "体液免疫和细胞免疫的因果链路有何异同？"

**考察层次**：跨专题的因果逻辑整合能力

---

### 类型4：逆向推导（Reverse Inference）

**定义**：从终端结果反推原因，训练诊断性思维。

**生成模板**：
```
"某实验中观察到{异常现象Z}，请推断最可能的{原因X}，并设计实验验证。"
"若检测到{中间产物M}大量积累而{下游产物N}减少，故障点最可能在哪个环节？"
```

**高中生物典型示例**：
- "实验发现某突变体在光照下ATP正常合成但NADPH不足，最可能哪个环节出了问题？"
- "若发现某患者的血糖持续偏高但胰岛素水平正常，可能的原因是什么？"
- "某细胞在减数分裂后出现非整倍体配子，最可能的原因是？"

**考察层次**：从结果反推机制的逻辑推理能力（实验设计题核心考点）

---

### 类型5：综合归因（Multi-chain Synthesis）

**定义**：将多个因果链并排对比或串联，考察系统思维。

**生成模板**：
```
"请绘制{现象P}的完整因果网络，说明{链路A}和{链路B}如何协同/拮抗。"
"从{层次1}到{层次2}再到{层次3}，各层次的因果逻辑如何衔接？"
```

**高中生物典型示例**：
- "从基因→mRNA→蛋白质→酶→代谢途径→性状，七步因果链中哪步最容易出问题？为什么？"
- "光合作用和呼吸作用在植物体内如何通过ATP/ADP和NADPH/NADP⁺的循环形成因果耦合？"
- "生态系统'种群→群落→生态系统'三个层次，每个层次的因果调节机制有什么不同？"

**考察层次**：综合分析与系统性思维能力

## 二、变式追问难度梯度

| 难度 | 对应变式类型 | 要求 | 预期回答时长 |
|------|------------|------|------------|
| ★☆☆ 基础 | 条件反转（单环节） | 能判断"有/无X的结果差异" | 30秒 |
| ★★☆ 进阶 | 环节跨越 + 条件反转（双环节） | 能解释中间机制的约束条件 | 1-2分钟 |
| ★★★ 挑战 | 逆向推导 + 类比迁移 | 能设计实验、跨专题类比 | 2-3分钟 |
| ★★★★ 压轴 | 综合归因 | 能串联多链路，形成因果网络 | 3-5分钟 |

---

## 三、自适应变式追问策略

根据用户回答质量，动态调整追问方向：

```
用户回答正确且深入
  → 升级难度（如基础→进阶→挑战→压轴）
  → 或切换到"类比迁移"检验跨专题理解

用户回答正确但浅层（只说了结论没说原因）
  → 追问"为什么？请具体解释{驱动机制}"
  → 保持当前难度，引导深挖

用户回答错误
  → 降级到更低难度变式
  → 回到因果链对应环节，重新解释
  → 提供"推导提示"而非直接给答案

用户回答不完整（漏了关键中间环节）
  → 追问"{漏掉的环节}在这里起什么作用？"
  → 补全因果链后再推进
```

## 五、按专题分类的变式追问示例

### 光合作用专题
| 变式类型 | 示例追问 |
|----------|---------|
| 条件反转 | 若类囊体膜上的ATP合酶被药物抑制，NADPH产量和O₂释放速率如何变化？ |
| 环节跨越 | C3植物在高温干旱时光合效率下降，有人说"因为气孔关闭导致光反应停止了"，对吗？ |
| 类比迁移 | 紫硫细菌只有PSI没有PSII，其电子传递链与高等植物有何不同？ |
| 逆向推导 | 测得某植物遮光后RuBP浓度先升后降，3-PGA浓度先降后升→推断光照和CO₂哪个先恢复？ |
| 综合归因 | 从"叶绿体结构→色素分布→电子传递→卡尔文循环→环境适应"五层因果，解释阳生叶和阴生叶的差异 |

### 呼吸作用专题
| 变式类型 | 示例追问 |
|----------|---------|
| 条件反转 | 若线粒体内膜对H⁺的通透性突然增大（解偶联），ATP产量和耗氧量如何变化？ |
| 环节跨越 | 无氧呼吸不经过电子传递链，为什么也需要NAD⁺？它与有氧呼吸的NAD⁺来源有何不同？ |
| 类比迁移 | 酵母菌在有氧和无氧条件下的代谢切换因果链，与人体肌肉细胞有何本质区别？ |
| 逆向推导 | 某细胞培养液中乳酸大量积累但ATP产量正常→推断O₂供给和线粒体功能的可能状态 |
| 综合归因 | 绘制葡萄糖→丙酮酸→乙酰CoA→TCA→氧化磷酸化的完整能量转化因果网络，标出每个节点的ATP消耗/产出 |

### 遗传学专题
| 变式类型 | 示例追问 |
|----------|---------|
| 条件反转 | 若某基因发生无义突变导致翻译提前终止，对其下游基因的表达有影响吗？（原核vs真核） |
| 环节跨越 | "基因突变一定导致性状改变"——从转录→翻译→蛋白质折叠→功能表现的角度评判这句话 |
| 类比迁移 | 原核生物的操纵子模型(lac operon)与真核生物的转录因子调控，因果逻辑有何本质区别？ |
| 逆向推导 | F₂出现9:7的分离比→推断控制该性状的基因关系和代谢通路（设计验证实验） |
| 综合归因 | 从"减数分裂→等位基因分离→配子类型→受精结合→子代表型"绘制完整遗传因果网络 |

### 免疫调节专题
| 变式类型 | 示例追问 |
|----------|---------|
| 条件反转 | 若Th细胞被HIV大量破坏，体液免疫和细胞免疫分别受到什么影响？哪个更严重？ |
| 环节跨越 | 为什么接种灭活疫苗后再次接触活病毒时抗体产生更快？中间的"记忆"环节是如何建立的？ |
| 类比迁移 | 过敏反应和自身免疫病在"免疫识别→效应→组织损伤"因果链上有何本质区别？ |
| 逆向推导 | 某患者反复感染但抗体检测正常→推断免疫系统的哪个环节可能异常？ |
| 综合归因 | 绘制"抗原进入→天然免疫→获得性免疫→免疫记忆"的完整因果时序图 |

## 六、追问链（Chase Chain）——连续追问的演化路径

当用户连续回答多个追问时，按以下路径递进：

### 路径 A：条件反转 → 环节跨越 → 逆向推导 → 综合归因
```
追问①(★): X被移除→Y如何变？
    ↓ 用户正确回答
追问②(★★): 为什么X移除后Z(下游远端)也变了？中间的B环节起什么作用？
    ↓ 用户正确回答
追问③(★★★): 若实验中观察到{W异常现象}，能反推出X的哪个具体步骤出问题了吗？
    ↓ 用户正确回答
追问④(★★★★): 将本因果链与{相似专题的因果链}做多维度对比
```

### 路径 B：逆向推导 → 条件反转 → 类比迁移（诊断型）
```
追问①(★★★): 异常结果Z → 推断可能原因 → 设计验证实验
    ↓
追问②(★): 若验证实验中抑制了你推断的那个环节，预测结果
    ↓
追问③(★★★): 同样的因果模式在{另一个生物过程}中出现过吗？怎么类比？
```

### 路径 C：快速诊断（选择题模式）
```
追问①(★): 给出A/B/C/D四个因果链描述，判断哪个正确
    ↓ 错误选项的阻断逻辑
追问②(★★): 错误选项的因果链在哪个环节断掉了？
    ↓
追问③(★★): 若修改错误选项的哪个条件，就能让它变成正确的？
```

## 七、变式追问质量自检清单

每次生成变式追问后，用以下清单自检：

- [ ] 每个追问都基于因果链中的具体环节（不凭空编造）
- [ ] 追问覆盖了因果链的不同位置（起点/中间/终点）
- [ ] 至少一个追问连接了跨专题知识（类比迁移或综合归因）
- [ ] 每道追问都有推导提示（不直接给答案）
- [ ] 参考要点包含关键机制术语（不是简单的是/否）
- [ ] 追问之间有递进关系（非平行罗列）

## 八、河北卷高频变式追问实战库

以下追问直接模拟河北卷近年真题的设问风格，可直接用于练习。

### 光合作用实战追问
1. "大棚种植时若长期阴雨天，为什么需要人工补光但不建议24小时照射？从光合与呼吸的因果关系分析。"
2. "C3植物在高温干旱时出现'午休现象'，若此时测定其叶肉细胞中RuBP和3-PGA的浓度变化，预测趋势并说明原因。"
3. "某突变体PSII功能正常但Cyt b6f复合体缺失，该突变体在光照下能否合成ATP？能否合成NADPH？能否释放O₂？"
4. "将正常植株从光下转移到暗处后，RuBP浓度先急剧上升后缓慢下降，请绘制浓度变化曲线并解释每一步的原因。"

### 呼吸作用实战追问
5. "人在剧烈运动时肌肉细胞中ATP/ADP比值如何变化？为什么ATP浓度不会降到零？请从细胞呼吸的调节机制分析。"
6. "线粒体DNA突变导致复合体I部分失活，细胞的耗氧量和ATP产量如何变化？若该突变发生在精子线粒体中，后代是否遗传？"
7. "酵母菌在葡萄糖培养液中先通气后密封，请预测培养液中乙醇浓度和酵母数量的变化曲线并解释。"
8. "为什么说'无氧呼吸第二阶段不产生ATP'？如果第二阶段不产能，那无氧呼吸的意义是什么？"

### 遗传学实战追问
9. "F₂出现9:7的分离比，请写出控制该性状的最可能代谢通路，并设计实验验证你的假设。"
10. "系谱图中III-2为红绿色盲患者，其父母表型正常，请判断该致病基因的传递路径并计算其妹妹为携带者的概率。"
11. "某基因mRNA序列如下: 5'-AUG CCU UAA GCU-3'，若第5位碱基C突变为A，翻译产物有何变化？"
12. "父亲患抗维生素D佝偻病(X显性)，母亲正常，请分析其子女的患病情况并解释为什么男女患病概率不同。"

### 实验设计实战追问
13. "设计实验验证'中耕松土能促进植物对矿质元素的吸收'，要求写出实验思路、预期结果和结论。"
14. "某同学设计了探究温度对酶活性影响的实验，但未考虑pH缓冲液的影响，请评价该实验设计的缺陷并改进。"
15. "请设计实验区分细胞呼吸产生的CO₂是来自糖酵解还是柠檬酸循环（提示：可用¹³C标记不同位置的葡萄糖）。"

## 九、变式追问质量自评矩阵

| 评价维度 | 1分(不足) | 3分(合格) | 5分(优秀) |
|---------|-----------|-----------|-----------|
| 覆盖因果链环节 | 只问一端 | 覆盖起点+终点 | 覆盖起点+中间+终点 |
| 难度梯度 | 全为同一难度 | 两个难度层次 | 至少三个难度层次 |
| 启发性 | 可直接回答是/否 | 需要简要推理 | 需要多步因果推导 |
| 跨专题连接 | 纯单专题 | 提及另一个专题 | 深度融合两个以上专题 |
| 与河北卷的一致性 | 套用通用模板 | 接近河北卷风格 | 完美匹配河北卷设问方式 |
| 推导提示质量 | 直接给答案思路 | 给关键突破点 | 仅给思维方向不泄露答案 |

| 维度 | 满分 | 标准 |
|------|------|------|
| 因果完整性 | 5 | 是否覆盖了完整的因果链（A→B→...→Z） |
| 机制准确性 | 5 | 驱动机制（酶/激素/基因/物理）表述是否精准 |
| 逻辑严密性 | 3 | 因果联结是否有跳跃或偷换概念 |
| 术语规范性 | 2 | 是否使用了正确的生物学名词 |

总分 15 分制。12 分以上为优秀，9 分及以上为合格。

# 遗传学题型速解手册 — 河北基因序列图专项

## 一、九种基因互作比例全解

### 9:3:3:1 — 标准独立遗传(两对基因无互作)

| 基因型 | 表型 | 因果 |
|--------|------|------|
| A_B_ (9/16) | 双显性状 | 两个显性基因产物独立发挥作用，表型叠加 |
| A_bb (3/16) | A显B隐 | 只有A基因产物 |
| aaB_ (3/16) | A隐B显 | 只有B基因产物 |
| aabb (1/16) | 双隐性状 | 两基因产物均缺 |

**河北陷阱**: 题目给的不是9:3:3:1但问你"是否符合自由组合"——先用测交或自交验证是否1:1:1:1或9:3:3:1

### 9:7 — 互补作用(双显性互补)

**代谢通路**: 前体→(酶A)→中间产物→(酶B)→终产物(有色/显性)
只有A_B_能完成完整通路产生终产物，其余三种基因型均在通路中途阻断。

**基因型-表型**: A_B_ = 显性性状(9)，其余 = 隐性性状(7)

**速判口诀**: "双显凑齐才显色，少一不可九比七"

### 9:3:4 — 隐性上位(aa抑制B/b表达)

**代谢通路**: 基因A控制色素合成，基因B控制色素沉积。aa纯合时完全阻断色素合成，B/b无法表现。
**基因型-表型**: A_B_ = 表型1(9)，A_bb = 表型2(3)，aa__ = 表型3(4)

**速判口诀**: "aa盖住B表达，三比率九三四分"

### 12:3:1 — 显性上位(A掩盖B/b表型)

**代谢通路**: A基因产物完全覆盖B基因的表型效应。只要A存在(A_)，无论B/b如何，表型都相同。
**基因型-表型**: A___ = 表型1(12)，aaB_ = 表型2(3)，aabb = 表型3(1)

**速判口诀**: "有A就盖B表现，十二三一显上位"

### 15:1 — 叠加作用(有A或有B即显)

**代谢通路**: A和B催化同一反应的不同步骤或平行通路，只要有任意一个就能产生终产物。
**基因型-表型**: 有A或有B = 表型1(15)，aabb = 表型2(1)

**速判口诀**: "有A有B都能显，只有双隐才不显"

### 9:6:1 — 积加作用(剂量效应)

**代谢通路**: A和B的产物独立贡献，可叠加。每个显性等位基因贡献等量效应。
**基因型-表型**: A_B_ = 完全表型(9)，A_bb或aaB_ = 部分表型(3+3=6)，aabb = 无表型(1)

**速判口诀**: "各献一份可叠加，双显全效单显半"

### 13:3 — 抑制基因

**代谢通路**: 抑制基因I的产物抑制显性基因A的表达，但自身无表型效应。只有当I不存在(ii)时A才能表达。
**基因型-表型**: I___或aabb或aaB_等 = 表型1(13)，iiA_ = 表型2(3)

**速判口诀**: "抑制基因一出手，十三比三显性躲"

### 1:4:6:4:1 — 数量性状(多基因加性)

**代谢通路**: 多个基因对同一性状有加性效应，每个显性等位基因贡献相等的量。表型呈连续分布。
**显性等位基因数**: 0(1):1(4):2(6):3(4):4(1)

### 3:1的各种变形速判

若F₂比例总和不是16的变体，可能是单基因控制但有其他修饰：
- 2:1 → 显性纯合致死
- 1:2:1 → 不完全显性(杂合子为中间表型)
- 3:1但其中一个表型有性别偏向 → 伴性遗传或从性遗传

## 二、伴性遗传六种模式详解

### X染色体隐性(如红绿色盲、血友病)

**遗传特征**:
- 男性患者远多于女性(男只需一个致病X，女需两个)
- 交叉遗传: 外祖父→(通过女儿携带)→外孙
- 双亲正常，儿子可能患病(母为携带者)

**基因型速判**:
- 患病男性: XᵃY
- 正常男性: XᴬY
- 患病女性: XᵃXᵃ(罕见)
- 携带者女性: XᴬXᵃ(表型正常)
- 正常女性: XᴬXᴬ

**概率计算关键**: 男性X来自母亲→若母亲为携带者(XᴬXᵃ)→儿子50%患病

### X染色体显性(如抗维生素D佝偻病)

**遗传特征**:
- 女性患者多于男性(女有两条X均可携带致病基因)
- 连续遗传(代代出现)
- 患病父亲(XᴬY)→女儿全患病(继承父亲的Xᴬ)，儿子全正常(继承父亲的Y)

### Y染色体遗传(如外耳道多毛症)

**遗传特征**:
- 仅限男性(限雄遗传)
- 父→子→孙垂直传递
- 无女性患者，无女性携带者

### 细胞质遗传(线粒体/叶绿体基因)

**遗传特征**:
- 母系遗传: 母亲患病→所有子女都可能患病
- 父亲患病→子女均正常(精子几乎不含细胞质)
- 不符合孟德尔比例

### 从性遗传(常染色体，表达受性别影响)

**本质**: 基因位于常染色体上，但表达受性激素等影响。如人类秃顶(男显女隐)。

### 限性遗传(性状仅在一种性别表现)

**本质**: 基因位于常染色体，但性状只在一种性别出现(如子宫发育、睾丸发育)。

## 三、系谱图四步推导法

### 步骤一: 判断显隐性

```
找"无中生有" → 隐性遗传
  例: 双亲正常(I-1□ I-2○)，子女患病(II-3■)

找"有中生无" → 显性遗传
  例: 双亲患病(I-1■ I-2●)，子女正常(II-3□)

两者均无 → 假设检验法
  先假设为常染色体隐性→逐一验证所有个体是否可能
```

### 步骤二: 判断染色体定位

```
常染色体隐性: 男女患病≈相等，散发出现
X染色体隐性: 男≫女，交叉遗传(外祖父→外孙)
X染色体显性: 女>男，连续遗传，父传女不传子
Y染色体: 仅男，父传子
```

### 步骤三: 逐一推导基因型

```
原则1: 隐性性状个体 → 基因型确定(aa 或 XᵃY)
原则2: 显性性状个体 → 若其子女有隐性→必为杂合(Aa)
原则3: 从确定个体推不确定个体
```

### 步骤四: 计算概率

```
步骤:
① 确定双亲基因型(含概率)
② 列出双亲产生的配子种类及比例
③ 画棋盘格或相乘组合概率
④ 只计算满足条件的基因型组合概率

河北陷阱: "已出生的孩子"vs"将要出生的孩子"
  → 已出生: 条件概率(在已知表型下计算)
  → 未出生: 直接概率
```

## 四、基因序列图专题(河北特色)

### 类型1: 单链序列→蛋白质推导

**例题**: 已知DNA模板链序列3'-TAC GGA TTC CGA ACT-5'
(1) 写出mRNA序列、编码链序列
(2) 翻译产物的氨基酸序列(查密码子表)
(3) 若第6位碱基T突变为C，产物如何变化？

**推导流程**:
① 编码链(有义链): 5'-ATG CCT AAG GCT TGA-3' (与mRNA一致，T→U)
② mRNA: 5'-AUG CCU AAG GCU UGA-3'
③ 翻译: Met-Pro-Lys-Ala-STOP (5肽→实际4肽，终止密码子不编码氨基酸)
④ 突变: T→C → 编码链变为5'-ATG CCC AAG GCT TGA-3' → Pro不变(CCU和CCC都是Pro)→同义突变

### 类型2: 限制酶图谱分析

**例题**: 某DNA片段全长5000bp，上有EcoRI位点2个，BamHI位点1个。EcoRI完全酶切得3个片段(2000+1800+1200bp)，EcoRI+BamHI双酶切得4个片段(2000+1000+800+1200bp)。绘制限制酶图谱。

**推导**: BamHI将1800bp片段切为1000+800→BamHI位于该1800bp片段内部，距一端1000bp

### 类型3: PCR引物设计

**设计原则**:
- 引物长度: 18-25bp
- GC含量: 40-60%
- 无自身互补(不发夹结构)
- 两引物Tm值相近(相差<5°C)
- 3'端最好是G或C(GC钳)

### 类型4: 电泳图谱分析

**条带解读**:
- 1条带 → 未酶切或无双酶切位点(环状→1条，线状→1条)
- 2条带 → 1个酶切位点
- n+1条带 → n个酶切位点(线状DNA)
- n条带 → n个酶切位点(环状DNA，如质粒)

## 五、遗传学计算速算公式库

| 问题 | 公式 | 说明 |
|------|------|------|
| 配子种类数 | 2ⁿ | n=杂合基因对数(完全显性) |
| F₂基因型种类 | 3ⁿ | 每对基因3种基因型(AA:Aa:aa) |
| F₂表型种类 | 2ⁿ | 每对基因2种表型(完全显性) |
| F₂纯合子比例 | (1/2)ⁿ | n对均纯合 |
| F₂杂合子比例 | 1-(1/2)ⁿ | 至少一对杂合 |
| 特定基因型: AaBbCc | (1/2)³=1/8 | 每对杂合概率1/2 |
| 测交子代表型比 | (1:1)ⁿ | 对独立基因 |
| 连续自交r代后杂合子 | (1/2)ʳ | 从F₁(Aa)开始 |
| 交换率 | 重组型/总配子×100% | < 50% |
| 遗传图距 | 交换率×100(cM) | 1%交换率=1cM |
| 双交换概率 | 交换率₁×交换率₂ | 独立事件 |

## 六、遗传学实验设计验证模板

### 验证基因分离定律

**测交法**: F₁(Aa) × 隐性纯合(aa)→子代表型比1:1→分离定律成立
**自交法**: F₁(Aa)自交→F₂表型比3:1→分离定律成立
**花粉鉴定法**: F₁花粉(含A或a)用特定染色法→两种花粉各50%

### 验证基因自由组合定律

**测交法**: F₁(AaBb) × 隐性纯合(aabb)→子代4种表型比1:1:1:1
**自交法**: F₁(AaBb)自交→F₂表型比9:3:3:1

### 判断基因在常染色体还是X染色体

**正反交法**:
- 正反交结果相同→常染色体
- 正反交结果不同→X染色体(或细胞质遗传)
**隐雌×显雄法**:
- 若子代雌全显、雄全隐→X染色体
- 若子代雌雄均显→常染色体

### 判断两对基因是否连锁

**测交**: F₁(AaBb) × aabb
- 若1:1:1:1→不连锁(独立)
- 若偏离1:1:1:1(亲本型远多于重组型)→连锁

## 七、河北卷遗传学易错速查

| 易错点 | 正确理解 |
|--------|---------|
| 分离定律的细胞学基础 | 同源染色体分离(减I后期)→等位基因分离，不是着丝粒分裂！|
| 自由组合的条件 | 两对基因位于非同源染色体上(不连锁) |
| 基因互作≠不遵循定律 | 互作只改变表型比例，不改变基因传递规律 |
| 交换率≠50% | 交换率<50%，50%以上按自由组合处理 |
| 测交≠检验所有 | 测交只检验F₁产生的配子种类和比例 |
| 多基因遗传≠等位 | 数量性状是多基因(非等位基因)加性效应 |
| 系谱优先判遗传方式 | 先确定遗传方式再推基因型，不要跳过第一步 |
| PCR扩增方向 | 5'→3'，两条引物扩增它们之间的片段 |

#### 2. 提问比回答更重要
**原理：** 苏格拉底式追问比直接给答案更能促进深度理解。

**应用到 bio-causal-reasoning：**
- 提供思考框架而非标准答案

**提问模板：**
1. "你认为这个过程的起点是什么？"
2. "如果X发生了变化，下一步会怎样？"
3. "你刚才说的和教材中的哪个概念矛盾？"
4. "能不能用一个实验来验证你的想法？"

#### 3. 情感连接是学习的前提

**应用到 bio-causal-reasoning：**

### 🚨 坑 1：半对半错（复合句陷阱）

**特征**：选项由多个分句组成，前半句对、后半句错，整句判错。

**典型踩坑**：ID-14（脂质题）B 选项："磷脂是构成生物膜的重要物质（✅），但并不是所有的细胞膜中都含磷脂（❌）"

**拆句法**（必须执行）：
1. 遇到 ≥2 个分句的选项，先拆成独立句子
2. 逐句判断正误
3. 任意一句错 → 整选项错

**高考高频复合句措辞**：
- "A 是 B，但 C…" → 拆成 A 句 + C 句
- "A 对 B 错" → 拆成 A 句 + B 句
- "既能…也能…" → 拆成两个独立判断

### 🚨 坑 2：概念混淆（最高频，71% 错误根因）

**易混淆概念对比表**：

| 混淆对 | 关键区别 | 高考常考 |
|---------|---------|---------|
| 载体蛋白 vs 通道蛋白 | 载体=构象改变（旋转门）；通道=亲水孔道（走廊） | 协助扩散两种类型 |
| NADH（呼吸）vs NADPH（光合） | NADH 产 ATP；NADPH 用于 C₃ 还原 | 电子传递链/光合暗反应 |
| 初步水解 vs 彻底水解 | 初步=核苷酸/氨基酸；彻底=磷酸+糖+碱基 | tRNA/核酸题高频 |
| 脱水缩合 vs 整体元素组成 | 脱水缩合不改变元素种类；整体=氨基酸+辅基 | 血红蛋白含 Fe |
| 自由扩散 vs 协助扩散 | 自由=脂溶性小分子；协助=需要载体蛋白 | 脂肪酸 vs 葡萄糖 |
| 主动运输 vs 被动运输 | 主动=逆浓度+耗 ATP；被动=顺浓度+不耗能 | 小肠上皮细胞 vs 红细胞 |

**应对策略**：遇到上表任意一对 → 先默念区别，再答题

### 🚨 坑 3：元素组成混淆

**各物质元素组成速查表**：

**陷阱**："脱水缩合可形成 X，故 X 的组成元素与单体相同" → ❌
- 脱水缩合只连接单体，不改变元素
- 但蛋白质整体可能含辅基（如血红素含 Fe），元素组成 ≠ 氨基酸元素组成
